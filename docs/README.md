## 🔄 文档更新机制

### 新增需求时
1. 更新 **需求总览文档** - 新增需求条目，分配需求编号
2. 更新 **功能详细说明文档** - 新增功能模块说明
3. 更新 **技术说明文档** - 新增技术实现说明
4. 更新 **测试说明文档** - 新增测试用例
5. 在 **开发记录** 中新增开发记录

### 版本发布时
1. 更新版本号
2. 更新需求追溯矩阵
3. 更新测试报告

## 🆕 最新功能更新 (v1.4.2)

### 资产卡片样式优化

**功能概述**: 统一首页资产统计卡片的样式，现有资产与初始资产保持一致的字体格式，收益率移除特殊背景色。

**修改内容**:
- 现有资产：移除高亮样式，与初始资产保持一致
- 收益率：移除背景高亮，使用默认底色
- 所有资产字段统一使用默认字体样式

**相关文件**:
- `lib/features/home/home_screen.dart`
