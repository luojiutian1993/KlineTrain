# K线训练营 - 功能详细说明文档

## 1. 首页模块 (HomeModule)

### 1.1 资产统计卡片 [_buildAssetStatsCard]
**功能描述**：展示用户当前资产状况的核心指标

**展示内容**：
| 指标 | 说明 |
|------|------|
| 初始资产 | 默认¥100,000 |
| 现有资产 | 当前账户总资产 |
| 总盈亏 | 绝对值和百分比 |
| 收益率 | 总收益率百分比 |
| 操作次数 | 历史交易总次数 |
| 操作天数 | 参与交易的天数 |
| 盈利次数 | 盈利交易的次数 |
| 成功率 | 盈利次数/总操作次数 |
| 最大盈利 | 单笔最大盈利 |
| 最大亏损 | 单笔最大亏损 |
| 最大回撤 | 历史最大回撤比例 |

**实现文件**：`lib/features/home/home_screen.dart`

### 1.2 收益率曲线 [_buildProfitCurve]

**展示内容**：

### 1.4 选股条件 [_buildStockSelection]
**功能描述**：提供选股条件筛选功能

**确认按钮 [_buildConfirmButton]**：
- 位置：选股条件卡片底部
- 样式：主题色背景，白色文字，圆角按钮
- 行为：点击后跳转到实战页面 (`context.go('/battle')`)

**实现文件**：`lib/features/home/home_screen.dart`

### 2.4 K线图表 [_buildKlineChart]
**功能描述**：渲染K线蜡烛图及均线

**组件**：`KlineChart` (lib/features/training/widgets/kline_chart.dart)

**K线数据模型**：
```dart
class KlineData {
  final DateTime date;
  final double open;   // 开盘价
  final double high;   // 最高价
  final double low;    // 最低价
  final double close;  // 收盘价
  final double volume; // 成交量
}
```

**均线计算 [_calculateMA]**：
- MA5：5日简单移动平均
- MA10：10日简单移动平均
- MA20：20日简单移动平均
- MA30：30日简单移动平均

**平滑渲染 [_drawSmoothLine]**：
- 使用二阶贝塞尔曲线 (`Path.quadraticBezierTo`)
- isAntiAlias = true 优化渲染
- 均线颜色：MA5黄色、MA10紫色、MA20橙色、MA30蓝色

**买卖点位标记 [_drawTradePoints]**：
```dart
class TradePoint {
  final int index;       // K线索引
  final double price;    // 交易价格
  final bool isBuy;      // true=买入，false=卖出
  final String label;    // 显示标签
  final DateTime date;   // 交易日期
}
```

**标记样式**：
| 类型 | 颜色 | 形状 |
|------|------|------|
| 买入 | 红色 | 向上三角形 |
| 卖出 | 绿色 | 向下三角形 |

**实现文件**：
- `lib/features/training/widgets/kline_chart.dart`
- `lib/features/battle/battle_screen.dart`

### 2.5 指标图表 [_buildIndicatorSelector] [_buildIndicatorSection]
**功能描述**：展示成交量、MACD等技术指标，每个指标下拉框与图表在同一区域内

**指标区域结构**：
```
┌─────────────────────────────────────────┐
│ 成交量 ▼   量                          │ ← 指标选择行
├─────────────────────────────────────────┤
│ [成交量柱状图]                          │ ← 指标图表
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│ MACD ▼   (12,26,9)                     │ ← 指标选择行
├─────────────────────────────────────────┤
│ [MACD柱状图]                            │ ← 指标图表
└─────────────────────────────────────────┘
```

**组件层级**：
- `_buildIndicatorSelector`：外层容器，包含两个指标区域
- `_buildIndicatorSection`：单个指标区域（下拉框 + 图表）
- `_buildSingleIndicatorChart`：根据选择的指标渲染对应图表

**指标下拉菜单**：
| 指标 | 参数显示 | 图表类型 |
|------|---------|---------|
| 成交量 | 量 | 柱状图 |
| MACD | (12,26,9) | 柱状图 |
| KDJ | (9,3,3) | 折线图 |
| RSI | (14) | 折线图 |
| BOLL | (20) | 轨道线图 |

**成交量柱状图 [_buildVolumeChart]**：
- 红色柱：上涨（收盘价>=开盘价）
- 绿色柱：下跌（收盘价<开盘价）
- 自动计算最大成交量并等比缩放
- safeMax处理避免除零错误

**MACD柱状图 [_buildMacdChart]**：
- DIF线：(EMA12 - EMA26)
- DEA线：DIF的EMA9
- MACD柱：DIF - DEA（正值红色，负值绿色）
- safeMax处理避免除零错误

**KDJ折线图 [_buildKdjChart]**：
- K线：黄色
- D线：紫色
- J线：红色
- 范围：0-100

**RSI折线图 [_buildRsiChart]**：
- RSI线：蓝色
- 范围：0-100

**BOLL轨道线图 [_buildBollChart]**：
- 上轨：橙色
- 中轨：紫色
- 下轨：绿色
- 基于收盘价计算布林带

**实现文件**：`lib/features/battle/battle_screen.dart`

### 2.8 买入页面 [_TradeDialog]
**功能描述**：设置买入价格和数量

**页面元素**：
| 元素 | 类型 | 默认值 |
|------|------|--------|
| 标题 | 文本 | "买入" |
| 价格输入 | TextField | 当前价 |
| 数量输入 | TextField | 可买入最大数量 |
| 1/3仓 | 按钮 | - |
| 1/2仓 | 按钮 | - |
| 全仓 | 按钮 | - |
| 买入金额 | 预览 | 实时计算 |
| 取消 | 按钮 | 关闭弹窗 |
| 确认 | 按钮 | 执行买入 |

**仓位计算逻辑 [_setPositionRatio]**：
- 1/3仓：`maxBuy * 0.33` 取整到100股
- 1/2仓：`maxBuy * 0.5` 取整到100股
- 全仓：`maxBuy` 取整到100股

**确认逻辑 [_executeBuy]**：
1. 更新持仓数量
2. 更新持仓成本
3. 扣减账户余额
4. 添加买入点位到交易记录
5. 显示成功提示

**实现文件**：`lib/features/battle/battle_screen.dart`

### 2.9 卖出页面 [_TradeDialog]
**功能描述**：设置卖出价格和数量

**页面元素**：
| 元素 | 类型 | 默认值 |
|------|------|--------|
| 标题 | 文本 | "卖出" |
| 价格输入 | TextField | 当前价 |
| 数量输入 | TextField | 全部持仓 |
| 1/3仓 | 按钮 | - |
| 1/2仓 | 按钮 | - |
| 全仓 | 按钮 | - |
| 卖出金额 | 预览 | 实时计算 |
| 预计盈亏 | 预览 | 实时计算 |
| 取消 | 按钮 | 关闭弹窗 |
| 确认 | 按钮 | 执行卖出 |

**预计盈亏计算**：
```
盈亏 = 卖出金额 - 持仓成本
     = 卖出价格 × 数量 - 成本价格 × 数量
```

**确认逻辑 [_executeSell]**：
1. 添加卖出点位到交易记录
2. 增加账户余额
3. 减少持仓数量
4. 持仓归零时重置成本
5. 显示成功提示

**实现文件**：`lib/features/battle/battle_screen.dart`

## 3. 数据模块

### 3.1 K线数据模型
**文件**：`lib/data/models/kline_model.dart`

```dart
class KlineModel {
  final String symbol;      // 股票代码
  final int timestamp;       // 时间戳
  final double open;        // 开盘价
  final double high;       // 最高价
  final double low;         // 最低价
  final double close;       // 收盘价
  final double volume;      // 成交量
  final double turnover;    // 成交额
}
```
