**数据库位置**（iOS模拟器）：
```
~/Library/Developer/CoreSimulator/Devices/[设备ID]/data/Containers/Data/Application/[应用ID]/Documents/kline_trainer.db
```

**为什么不在项目路径下**：
- iOS/macOS应用使用沙箱机制

- 删除应用时自动清理数据

### 2.4 K线图表 [_buildKlineChart]
**功能描述**：渲染K线蜡烛图及均线

**组件**：`KlineChart` (lib/features/training/widgets/kline_chart.dart)

**K线数据模型**：
```dart
class KlineData {
  final DateTime date;
  final double open;   // 开盘价
  final double high;   // 最高价
  final double low;    // 最低价
  final double close;  // 收盘价
  final double volume; // 成交量

```

**均线计算 [_calculateMA]**：
- MA5：5日简单移动平均
- MA10：10日简单移动平均

**平滑渲染 [_drawSmoothLine]**：
- 使用二阶贝塞尔曲线 (`Path.quadraticBezierTo`)
- isAntiAlias = true 优化渲染
- 均线颜色：MA5黄色、MA10紫色、MA20橙色、MA30蓝色

**买卖点位标记 [_drawTradePoints]**：

### 5.6 指标图表 [_buildIndicatorSelector] [_buildIndicatorSection]
**功能描述**：展示成交量、MACD等技术指标，每个指标下拉框与图表在同一区域内

**指标下拉菜单**：
| 指标 | 参数显示 | 图表类型 |
|------|---------|---------|
| 成交量 | 量 | 柱状图 |
| MACD | (12,26,9) | 柱状图 |
| KDJ | (9,3,3) | 折线图 |
| RSI | (14) | 折线图 |
| BOLL | (20) | 轨道线图 |
| **CCI** | (14) | 折线图 |
| **OBV** | - | 折线图 |
| **DMI** | (14) | 折线图 |
| **DMA** | (10,50,10) | 折线图 |
| **BBI** | (3,6,12,24) | 折线图 |

**CCI指标 [_buildCciChart]**：
- 算法：CCI = (TP - SMA_TP) / (0.015 × MD)
- 参数：N = 14
- 超买区域：> 100
- 超卖区域：< -100
- 颜色：蓝色

**OBV指标 [_buildObvChart]**：
- 算法：如果收盘>昨日收盘，OBV累加成交量；反之累减
- 颜色：蓝色
- 数据范围检查：避免空数据RangeError

**DMI指标 [_buildDmiChart]**：
- 算法：计算+DI、-DI、ADX
- 参数：N = 14
- +DI颜色：红色
- -DI颜色：绿色
- ADX颜色：蓝色
- 数据对齐：ADX数据可能比+DI/-DI短，需要填充默认值

**DMA指标 [_buildDmaChart]**：
- 算法：DMA = 快线SMA - 慢线SMA，AMA = DMA的EMA
- 参数：短周期=10，长周期=50，signal=10
- DMA颜色：红色
- AMA颜色：蓝色
- 范围检查：数据范围为零时使用默认范围，避免horizontalInterval=0错误

**BBI指标 [_buildBbiChart]**：
- 算法：BBI = (MA3 + MA6 + MA12 + MA24) / 4
- 颜色：橙色

**实现文件**：`lib/features/battle/battle_screen.dart`

## 2.8 指标同步联动 [_buildIndicatorSection]
**功能描述**：指标图表随K线滑动缩放同步变化

**同步机制**：
1. **数据范围同步**：指标数据范围随K线可视区间自动调整
2. **时间轴对齐**：指标图表时间轴与K线完全对齐，滑动时同步移动
3. **视觉缩放**：指标数值显示随K线缩放比例调整

**联动流程**：
```
K线滑动/缩放 → 更新可视区间 → 过滤指标数据 → 重绘指标图表
```

**关键实现**：
```dart
// 同步可视区间
void _updateVisibleRange() {
  final startIndex = _calculateStartIndex();
  final endIndex = _calculateEndIndex();
  _visibleIndicatorData = _allIndicatorData.sublist(startIndex, endIndex);
}

// 动态计算指标数据范围
void _updateIndicatorRange() {
  _indicatorMin = _visibleIndicatorData.reduce(min);
  _indicatorMax = _visibleIndicatorData.reduce(max);
}
```

## 2.9 K线可视区域优化 [_buildKlineChart]
**功能描述**：实战页面初始化时显示包含当前训练日的K线，点击下一步时自动滑动保持当前日可视

**状态变量**：
| 变量 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `_currentDayIndex` | int | 0 | 当前训练天数索引 |
| `_visibleStartIndex` | int | 0 | 可视区域起始索引 |
| `_visibleKlineCount` | int | 20 | 可视K线数量 |

**实现逻辑**：
```dart
// 初始化时计算可视区域
void _initVisibleRange() {
  final endIndex = _currentDayIndex + 1;
  _visibleStartIndex = max(0, endIndex - _visibleKlineCount);
}

// 点击下一步时更新
void _nextDay() {
  _currentDayIndex++;
  final endIndex = _currentDayIndex + 1;
  _visibleStartIndex = max(0, endIndex - _visibleKlineCount);
}
```

**验收标准**：
- 初始化时当前训练日显示在K线图最右侧
- 点击下一步时K线自动向左滑动
- 当前训练日始终保持在可视范围内

**实现文件**：`lib/features/battle/battle_screen.dart`

## 5. 指标计算引擎

### 5.1 指标计算工具类
**文件**：`lib/data/utils/indicator_calculator.dart`

**核心方法**：
| 方法 | 返回值 | 说明 |
|------|--------|------|
| `calculateMACD()` | `MACDResult` | MACD指标（DIF/DEA/MACD柱） |
| `calculateKDJ()` | `KDJResult` | KDJ随机指标 |
| `calculateRSI()` | `RSIResult` | RSI相对强弱指标 |
| `calculateBoll()` | `BollResult` | 布林带指标 |
| `calculateWR()` | `WRResult` | 威廉指标 |
| `calculateMA()` | `List<double>` | 简单移动平均线 |

### 5.2 MACD指标算法
```dart
// 计算公式
DIF = EMA12 - EMA26
DEA = EMA(DIF, 9)
MACD = 2 * (DIF - DEA)

// EMA计算公式
EMA[t] = close[t] * k + EMA[t-1] * (1-k)
k = 2 / (N + 1)
```

### 5.3 KDJ指标算法
```dart
// 计算公式
RSV = (close - LLV(low, 9)) / (HHV(high, 9) - LLV(low, 9)) * 100
K = 2/3 * K_prev + 1/3 * RSV
D = 2/3 * D_prev + 1/3 * K
J = 3 * K - 2 * D
```

### 5.4 RSI指标算法
```dart
// 计算公式
RSI = 100 - 100 / (1 + RS)
RS = 平均上涨幅度 / 平均下跌幅度
```

### 5.5 BOLL布林带算法
```dart
// 计算公式
MB = SMA(close, 20)  // 中轨
MD = SQRT(SUM((close-MB)^2)/N)  // 标准差
UP = MB + 2 * MD  // 上轨
DN = MB - 2 * MD  // 下轨
```
