### 2.4 K线图表 [_buildKlineChart]
**功能描述**：渲染K线蜡烛图及均线

**组件**：`KlineChart` (lib/features/training/widgets/kline_chart.dart)

**K线数据模型**：
```dart
class KlineData {
  final DateTime date;
  final double open;   // 开盘价
  final double high;   // 最高价
  final double low;    // 最低价
  final double close;  // 收盘价
  final double volume; // 成交量
}
```

**均线计算 [_calculateMA]**：
- MA5：5日简单移动平均
- MA10：10日简单移动平均
- MA20：20日简单移动平均
- MA30：30日简单移动平均

**平滑渲染 [_drawSmoothLine]**：
- 使用二阶贝塞尔曲线 (`Path.quadraticBezierTo`)
- isAntiAlias = true 优化渲染
- 均线颜色：MA5黄色、MA10紫色、MA20橙色、MA30蓝色

**买卖点位标记 [_drawTradePoints]**：
```dart
class TradePoint {
  final int index;       // K线索引

  final bool isBuy;      // true=买入，false=卖出
  final String label;    // 显示标签
  final DateTime date;   // 交易日期
}
```

**标记样式**：
| 类型 | 颜色 | 形状 |
|------|------|------|
| 买入 | 红色 | 向上三角形 |
| 卖出 | 绿色 | 向下三角形 |

### 5.6 指标图表 [_buildIndicatorSelector] [_buildIndicatorSection]
**功能描述**：展示成交量、MACD等技术指标，每个指标下拉框与图表在同一区域内

**指标下拉菜单**：
| 指标 | 参数显示 | 图表类型 |
|------|---------|---------|
| 成交量 | 量 | 柱状图 |
| MACD | (12,26,9) | 柱状图 |
| KDJ | (9,3,3) | 折线图 |
| RSI | (14) | 折线图 |
| BOLL | (20) | 轨道线图 |
| **CCI** | (14) | 折线图 |
| **OBV** | - | 折线图 |
| **DMI** | (14) | 折线图 |
| **DMA** | (10,50,10) | 折线图 |
| **BBI** | (3,6,12,24) | 折线图 |

**CCI指标 [_buildCciChart]**：
- 算法：CCI = (TP - SMA_TP) / (0.015 × MD)
- 参数：N = 14
- 超买区域：> 100
- 超卖区域：< -100
- 颜色：蓝色

**OBV指标 [_buildObvChart]**：
- 算法：如果收盘>昨日收盘，OBV累加成交量；反之累减
- 颜色：蓝色
- 数据范围检查：避免空数据RangeError

**DMI指标 [_buildDmiChart]**：
- 算法：计算+DI、-DI、ADX
- 参数：N = 14
- +DI颜色：红色
- -DI颜色：绿色
- ADX颜色：蓝色
- 数据对齐：ADX数据可能比+DI/-DI短，需要填充默认值

**DMA指标 [_buildDmaChart]**：
- 算法：DMA = 快线SMA - 慢线SMA，AMA = DMA的EMA
- 参数：短周期=10，长周期=50，signal=10
- DMA颜色：红色
- AMA颜色：蓝色
- 范围检查：数据范围为零时使用默认范围，避免horizontalInterval=0错误

**BBI指标 [_buildBbiChart]**：
- 算法：BBI = (MA3 + MA6 + MA12 + MA24) / 4
- 颜色：橙色

**实现文件**：`lib/features/battle/battle_screen.dart`

### 2.9 训练记录数据库保存 [_saveTrainingToDatabase]
**功能描述**：训练完成后自动保存训练记录到SQLite数据库

**数据表**：
1. **training_sessions表**：保存训练会话信息
2. **trades表**：保存每笔交易记录

**核心方法**：
```dart
Future<void> _saveTrainingToDatabase(
  double initialBalance,
  double totalAssets,
  double profit,
  double profitRate,
  double stockIncrease,
) async {
  // 1. 创建训练会话
  final sessionId = await dbService.trainingDao.createSession(
    TrainingSessionsCompanion(...),
  );

  // 2. 保存每笔交易
  for (final point in _tradePoints) {
    await dbService.trainingDao.addTrade(
      TradesCompanion(...),
    );
  }
}
```

**辅助方法**：
- `_calculateWinCount()`：计算盈利交易次数
- `_calculateWinRate()`：计算胜率

## 3. 数据模块

### 3.1 K线数据模型
**文件**：`lib/data/models/kline_model.dart`

```dart
class KlineModel {
  final String symbol;      // 股票代码
  final int timestamp;       // 时间戳
  final double open;        // 开盘价
  final double high;       // 最高价
  final double low;         // 最低价
  final double close;       // 收盘价
  final double volume;      // 成交量
  final double turnover;    // 成交额
}
```

### 3.2 K线仓库
**文件**：`lib/data/repositories/kline_repository.dart`

**数据获取优先级**：
1. 本地SQLite数据库
2. API接口
3. 模拟数据兜底

**核心方法**：

## 5. 指标计算引擎

### 5.1 指标计算工具类
**文件**：`lib/data/utils/indicator_calculator.dart`

**核心方法**：
| 方法 | 返回值 | 说明 |
|------|--------|------|
| `calculateMACD()` | `MACDResult` | MACD指标（DIF/DEA/MACD柱） |
| `calculateKDJ()` | `KDJResult` | KDJ随机指标 |
| `calculateRSI()` | `RSIResult` | RSI相对强弱指标 |
| `calculateBoll()` | `BollResult` | 布林带指标 |
| `calculateWR()` | `WRResult` | 威廉指标 |
| `calculateMA()` | `List<double>` | 简单移动平均线 |

### 5.2 MACD指标算法
```dart
// 计算公式
DIF = EMA12 - EMA26
DEA = EMA(DIF, 9)
MACD = 2 * (DIF - DEA)

// EMA计算公式
EMA[t] = close[t] * k + EMA[t-1] * (1-k)
k = 2 / (N + 1)
```

### 5.3 KDJ指标算法
```dart
// 计算公式
RSV = (close - LLV(low, 9)) / (HHV(high, 9) - LLV(low, 9)) * 100
K = 2/3 * K_prev + 1/3 * RSV
D = 2/3 * D_prev + 1/3 * K
J = 3 * K - 2 * D
```

### 5.4 RSI指标算法
```dart
// 计算公式
RSI = 100 - 100 / (1 + RS)
RS = 平均上涨幅度 / 平均下跌幅度
```

### 5.5 BOLL布林带算法
```dart
// 计算公式
MB = SMA(close, 20)  // 中轨
MD = SQRT(SUM((close-MB)^2)/N)  // 标准差
UP = MB + 2 * MD  // 上轨
DN = MB - 2 * MD  // 下轨
```
