# K线展示优化需求

> **版本**: v1.0
> **创建日期**: 2026-05-27
> **最后更新**: 2026-05-27
> **状态**: 已发布

---

## 1. 背景与价值

**为什么做**：当前K线图表展示效果不够清晰，用户难以准确识别K线形态和关键价格水平。同时缺少开盘价参考线和成本价参考线，用户无法直观判断当前价格与关键价格的关系。买卖点位标记信息过多，影响图表清晰度。

**做成什么样**：优化K线渲染效果使其更清晰锐利，添加开盘价虚线和成本价虚线作为参考，简化买卖点标记仅显示B/S标识。

**关联**：项目核心功能优化 — 提升训练页面用户体验

---

## 2. 范围

### 本次做

- K线图表渲染优化：提升图像清晰度和锐利度
- 新增蓝色开盘价虚线：贯穿整个K线图，随每天开盘动态变化
- 新增红色成本价虚线：贯穿整个K线图，根据交易记录计算平均成本
- 修改买卖点展示：仅显示B/S标识，移除价格和数量信息

### 本次不做

- 不修改K线数据获取逻辑
- 不修改交易功能核心逻辑
- 不修改页面布局结构

## 4. 需求详述

### 4.1 核心流程

1. 用户进入实战页面
2. K线图表渲染优化后的清晰效果
3. 蓝色开盘价虚线显示当前日开盘价
4. 红色成本价虚线显示持仓成本（如有持仓）
5. 买卖点仅显示B/S标识

### 4.2 交互规则

- K线图缩放和滑动时，开盘价虚线和成本价虚线随之同步移动
- 每天推进K线时，开盘价虚线动态更新为当日开盘价
- 成本价虚线在清仓后自动隐藏
- 买卖点仅显示B/S字母标识，不显示价格和数量

### 4.3 业务规则

| 编号 | 规则 |
|------|------|
| BR-001 | K线渲染需开启抗锯齿，蜡烛图边框清晰锐利 |
| BR-002 | 开盘价虚线为蓝色(#3B82F6)、横向贯穿整个图表区域 |
| BR-003 | 开盘价虚线在每天K线推进时动态更新为当日开盘价 |
| BR-004 | 成本价虚线为红色(#EF4444)、横向贯穿整个图表区域 |
| BR-005 | 成本价 = Σ(买入价格 × 买入数量) / Σ买入数量，每次买卖后重新计算 |
| BR-006 | 清仓后（持仓数量为0），成本价虚线不显示 |
| BR-007 | 买入点标记为红色B，卖出点标记为绿色S |
| BR-008 | 买卖点从当天K线最高价（买入）或最低价（卖出）位置直接引出蓝色垂直虚线，长度固定 |
| BR-009 | B/S标识显示在垂直虚线尽头，带白色背景和彩色边框 |
| BR-010 | 买卖点不显示三角框标注 |

## 5. 验收标准

### 5.1 正常流程 (Happy Path)

- [ ] **AC-001**: K线图表清晰锐利
  - **Given** 用户进入实战页面
  - **When** K线图表渲染完成
  - **Then** K线蜡烛图边缘清晰锐利，无模糊锯齿

- [ ] **AC-002**: 开盘价虚线显示
  - **Given** 用户进入实战页面，有K线数据
  - **When** 页面加载完成
  - **Then** 蓝色横向虚线贯穿整个K线图，标注当日开盘价

- [ ] **AC-003**: 开盘价虚线动态更新
  - **Given** 用户在实战页面
  - **When** 点击"下一步"推进到下一天
  - **Then** 蓝色开盘价虚线移动到新一天的开盘价位置

- [ ] **AC-004**: 成本价虚线显示
  - **Given** 用户有持仓
  - **When** 查看K线图表
  - **Then** 红色横向虚线贯穿整个K线图，代表当前持仓成本

- [ ] **AC-005**: 成本价动态计算
  - **Given** 用户执行买入或卖出操作
  - **When** 交易完成
  - **Then** 成本价重新计算并更新红色虚线位置

- [ ] **AC-006**: 成本价虚线隐藏
  - **Given** 用户持仓数量为0（清仓状态）
  - **When** 查看K线图表
  - **Then** 红色成本价虚线不显示

- [ ] **AC-007**: 买卖点简化显示
  - **Given** 用户有交易记录
  - **When** 查看K线图表
  - **Then** 买入点显示红色B，卖出点显示绿色S

- [ ] **AC-008**: 买卖点虚线延展
  - **Given** 用户有交易记录
  - **When** 查看K线图表
  - **Then** 买卖点从价格位置向上/下延展垂直虚线，仅显示B/S标识，无价格和数量信息

### 5.2 边界与异常 (Edge & Error Cases)

- [ ] **AC-009**: 无持仓时成本线隐藏
  - **Given** 用户初始状态（无任何交易）
  - **When** 进入实战页面
  - **Then** 红色成本价虚线不显示

- [ ] **AC-010**: 多次买卖后成本计算正确
  - **Given** 用户多次买入和卖出同一股票
  - **When** 查看持仓信息和K线图
  - **Then** 成本价虚线位置与持仓成本一致

- [ ] **AC-011**: 部分清仓后成本线仍显示
  - **Given** 用户卖出部分持仓但仍有剩余
  - **When** 查看K线图表
  - **Then** 红色成本价虚线继续显示

### 5.3 业务规则验证 (Business Rules)

- [ ] **AC-012**: K线渲染质量验证 ← BR-001
  - **Given** K线图表加载完成
  - **When** 放大查看K线细节
  - **Then** K线蜡烛图边缘清晰，抗锯齿效果明显

- [ ] **AC-013**: 开盘价虚线样式验证 ← BR-002
  - **Given** K线图表加载完成
  - **When** 查看图表
  - **Then** 开盘价虚线为蓝色虚线，贯穿整个图表区域

- [ ] **AC-014**: 成本价虚线样式验证 ← BR-004
  - **Given** 用户有持仓
  - **When** 查看图表
  - **Then** 成本价虚线为红色虚线，贯穿整个图表区域

- [ ] **AC-015**: 买卖点标记样式验证 ← BR-007, BR-008
  - **Given** 用户有交易记录
  - **When** 查看K线图表
  - **Then** 买入点显示红色B，卖出点显示绿色S，无价格数量信息
