# 买入卖出仓位设置功能需求文档

## 1. 功能概述

为买入和卖出交易弹窗添加仓位快捷选择功能，支持用户快速选择预设仓位比例，并可通过编辑功能自定义仓位配置。

## 2. 功能需求

### 2.1 仓位按钮展示

| 需求点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| AC-001 | 买入页面仓位行按以下顺序展示：全仓、1/2仓、1/3仓、1/4仓、2/3仓、编辑 | 高 |
| AC-002 | 卖出页面仓位行按以下顺序展示：全仓、1/2仓、1/3仓、1/4仓、2/3仓、编辑 | 高 |
| AC-003 | 点击仓位按钮自动计算并填充对应数量 | 高 |

### 2.2 仓位数学计算

| 需求点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| AC-004 | 全仓 = 1.0（全部可用资金/持仓） | 高 |
| AC-005 | 1/2仓 = 0.5（一半可用资金/持仓） | 高 |
| AC-006 | 1/3仓 = 1/3（三分之一可用资金/持仓） | 高 |
| AC-007 | 1/4仓 = 0.25（四分之一可用资金/持仓） | 高 |
| AC-008 | 2/3仓 = 2/3（三分之二可用资金/持仓） | 高 |
| AC-009 | 买入时数量按100股为单位取整 | 高 |
| AC-010 | 卖出时数量按100股为单位取整 | 高 |

### 2.3 仓位设置弹窗

| 需求点 | 描述 | 优先级 |
| :--- | :--- | :--- |
| AC-011 | 点击编辑按钮弹出仓位设置弹窗 | 高 |
| AC-012 | 弹窗包含买入仓位和卖出仓位两个标签页 | 高 |
| AC-013 | 支持拖动仓位按钮排序 | 高 |
| AC-014 | 支持添加新仓位（最多12个） | 高 |
| AC-015 | 支持删除已有仓位（至少保留1个） | 高 |
| AC-016 | 支持重置为默认值 | 高 |
| AC-017 | 支持保存仓位配置 | 高 |
| AC-018 | 显示"买入时不弹确认框"选项 | 中 |

## 3. 业务规则

| 规则编号 | 规则描述 |
| :--- | :--- |
| BR-001 | 买入时：可买数量 = (账户余额 / 当前价格 / 100).floor() * 100 * 仓位比例 |
| BR-002 | 卖出时：可卖数量 = 持仓数量 * 仓位比例 |
| BR-003 | 所有数量计算结果需按100股取整 |
| BR-004 | 仓位数量限制：最少1个，最多12个 |
| BR-005 | 默认按第1个仓位买入（当未选择其他仓位时） |

## 4. 界面原型描述

### 4.1 交易弹窗仓位行

```
仓位: [全仓] [1/2仓] [1/3仓] [1/4仓] [2/3仓] [编辑]
```

### 4.2 仓位设置弹窗

- 标题：仓位设置
- 标签页：买入仓位 / 卖出仓位
- 提示文字：拖动按钮可以排序，最多可拥有12个仓位，默认按第1个仓位买入
- 仓位列表：可拖动排序的仓位按钮，每个按钮右侧有删除图标
- 添加按钮：+（当仓位数量<12时显示）
- 选项：买入时不弹确认框（复选框）
- 按钮：默认值 / 保存买入仓位

## 5. 验收标准

| 验收项 | 验证方法 | 预期结果 |
| :--- | :--- | :--- |
| AC-001 | 打开买入弹窗 | 仓位按钮按全仓、1/2仓、1/3仓、1/4仓、2/3仓、编辑顺序显示 |
| AC-002 | 打开卖出弹窗 | 仓位按钮按全仓、1/2仓、1/3仓、1/4仓、2/3仓、编辑顺序显示 |
| AC-003 | 点击任一仓位按钮 | 数量输入框自动填充对应比例的数量 |
| AC-004 | 点击全仓按钮 | 数量 = 最大可买/卖数量 |
| AC-005 | 点击1/2仓按钮 | 数量 = 最大可买/卖数量 * 0.5 |
| AC-006 | 点击1/3仓按钮 | 数量 = 最大可买/卖数量 * 1/3 |
| AC-007 | 点击1/4仓按钮 | 数量 = 最大可买/卖数量 * 0.25 |
| AC-008 | 点击2/3仓按钮 | 数量 = 最大可买/卖数量 * 2/3 |
| AC-009 | 买入时计算数量 | 结果按100股取整 |
| AC-010 | 卖出时计算数量 | 结果按100股取整 |
| AC-011 | 点击编辑按钮 | 弹出仓位设置弹窗 |
| AC-012 | 在仓位设置弹窗中切换标签 | 显示对应标签页的仓位配置 |
| AC-013 | 拖动仓位按钮到新位置 | 仓位顺序更新 |
| AC-014 | 点击+按钮（仓位<12） | 添加新仓位（1/N仓格式） |
| AC-015 | 点击仓位右侧×按钮（仓位>1） | 删除该仓位 |
| AC-016 | 点击默认值按钮 | 恢复默认仓位配置 |
| AC-017 | 点击保存按钮 | 保存配置并关闭弹窗 |
| AC-018 | 勾选"买入时不弹确认框" | 复选框选中状态改变 |
