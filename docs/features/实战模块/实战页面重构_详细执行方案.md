**目标**：将 BattleScreen 改造为纯容器，使用新组件

1. 添加 BattleProvider 的 import

5. 修改 `initState()` 调用 Provider 的初始化方法

**关键修改**：
```dart
// 修改前
class _BattleScreenState extends ConsumerState<BattleScreen> {
  String _currentSymbol = 'SH600000';
  List<KlineModel> _allKlineData = [];

  @override
  void initState() {
    _loadKlineData();
  }

  Widget build(BuildContext context) {
    return _buildKlineChart();
  }
}

// 修改后
class _BattleScreenState extends ConsumerState<BattleScreen> {
  @override
  void initState() {
    super.initState();
    WidgetsBinding.instance.addPostFrameCallback((_) {
      ref.read(battleProvider.notifier).initializeRandom();
    });
  }

  Widget build(BuildContext context) {
    return const KlineChartWidget();
  }
}
```
