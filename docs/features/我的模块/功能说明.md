# "我的"页面功能文档

## 1. 功能概述

"我的"页面是 K线训练营应用的用户个人中心，提供用户信息管理、训练数据统计、功能导航等核心功能。

---

## 2. 主题颜色

"我的"页面采用与应用一致的棕色/琥珀色主题，确保与首页、实战等页面视觉统一。

### 2.1. 主题色系

| 用途 | 颜色 | 色值 | 说明 |
|------|------|------|------|
| 主色调 | AppTheme.accent | RGB(150, 90, 55) | 用于用户信息卡片背景、图标等 |
| 次要色 | AppTheme.accentSoft | 透明度38% | 用于图标背景等 |
| 成功色 | AppTheme.green | RGB(60, 150, 80) | 正收益率显示 |
| 警告色 | AppTheme.red | RGB(180, 60, 50) | 负收益率、退出按钮等 |
| 背景色 | AppTheme.bg | RGB(247, 245, 243) | 页面背景 |
| 卡片色 | AppTheme.surface | RGB(252, 251, 250) | 卡片背景 |
| 边框色 | AppTheme.border | RGB(227, 224, 220) | 卡片边框 |

### 2.2. 设计特点

- **卡片圆角**：统一使用 16px 圆角，与其他页面保持一致
- **阴影效果**：用户信息卡片添加阴影，增强层次感
- **渐变背景**：用户信息卡片使用主题色渐变
- **图标样式**：图标使用统一的 accent 颜色，配以柔和背景

## 3. 功能模块

### 3.1. 用户信息卡片

**位置**: 页面顶部

**功能描述**:
- 展示用户头像（圆形占位符，支持点击更换）
- 展示用户昵称（支持点击修改）
- 展示脱敏手机号（格式：138****8001）
- 展示会员等级徽章

**交互设计**:
| 操作 | 触发方式 | 反馈 |
|------|----------|------|
| 更换头像 | 点击头像 | 弹出底部选择菜单（相册/拍照） |
| 修改昵称 | 点击昵称旁编辑图标 | 弹出修改对话框 |

### 3.2. 训练统计卡片

**位置**: 用户信息卡片下方

**功能描述**:
- 训练次数：展示用户累计训练天数
- 总收益率：展示累计盈亏百分比（红色为正，绿色为负）
- 交易胜率：展示交易获胜比例
- 学习进度：展示课程完成百分比

**数据来源**:
- 从数据库 `users` 表读取 `totalTrainingDays`、`totalProfit`、`overallWinRate` 等字段

### 3.3. 功能菜单区

**位置**: 统计卡片下方

**菜单列表**:

| 菜单名称 | 图标 | 目标页面 | 说明 |
|----------|------|----------|------|
| 自选管理 | ⭐ | /mine/favorites | 管理关注品种 |
| 训练记录 | 📊 | /mine/training-history | 查看历史训练 |
| 学习进度 | 📚 | /mine/learning-progress | 课程学习进度 |
| 消息通知 | 🔔 | /mine/notifications | 系统通知（带未读数） |
| 意见反馈 | 💬 | /mine/feedback | 用户反馈提交 |
| 帮助中心 | ❓ | /mine/help-center | 常见问题解答 |
| 关于我们 | ℹ️ | 弹窗 | 应用信息 |

### 3.4. 设置入口

**位置**: 用户信息卡片右上角

**功能描述**:
- 点击齿轮图标进入设置页面
- 设置页面包含显示设置、交易设置等

### 3.5. 退出登录

**位置**: 页面底部

**功能描述**:
- 显示"退出登录"按钮
- 点击后弹出确认对话框
- 确认后清除登录状态，跳转登录页面

## 4. 用户认证流程

### 4.1. 注册流程

```
注册页面 → 输入手机号 → 获取验证码 → 输入验证码 → 输入密码 → 点击注册 → 保存数据库 → 跳转我的页面
```

**验证规则**:
- 手机号必须为11位数字
- 验证码必须为6位数字
- 密码至少6位

### 4.2. 登录流程

```
登录页面 → 输入手机号 → 输入密码 → 点击登录 → 验证数据库 → 跳转我的页面
```

---

## 5. 数据流向

```
注册/登录 → AuthProvider.register/login() → UserRepository.saveUserToDatabase()
                                                    ↓
                                           SQLite数据库(users表)
                                                    ↓
                                  UserRepository.getCurrentUser() → UserProvider
                                                    ↓
                                               MineScreen展示
```
