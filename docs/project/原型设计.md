# K线训练营 - 原型设计文档

## 1. 设计概述

### 1.1 设计规范

| 项目 | 规范 |
|------|------|
| **设计尺寸** | iPhone 14 Pro (390×844pt) |
| **色彩系统** | OKLCH色彩空间，暖灰色调 |
| **字体系统** | Display: Iowan Old Style / Body: System UI / Mono: SF Mono |
| **圆角规范** | 卡片 18px / 按钮 10px / 标签 999px |
| **间距系统** | 基础单位 4px，主要使用 8px/12px/16px/20px |

### 1.2 色彩定义

```css
:root {
  --bg: oklch(97% 0.012 80);           /* 背景色 - 暖灰 */
  --surface: oklch(99% 0.005 80);      /* 卡片表面 */
  --fg: oklch(20% 0.02 60);            /* 主文字 */
  --muted: oklch(48% 0.015 60);        /* 次要文字 */
  --border: oklch(89% 0.012 80);       /* 边框 */
  --accent: oklch(58% 0.16 35);        /* 强调色 - 暖棕 */
  --accent-soft: oklch(58% 0.08 35 / 0.15);
  --green: oklch(58% 0.16 145);        /* 涨 - 绿 */
  --red: oklch(58% 0.16 25);           /* 跌 - 红 */
}
```

### 3.2 功能组件

#### 3.2.1 资产统计卡片

| 字段 | 说明 | 样式 |
|------|------|------|
| 初始资产 | 训练开始时的资金 | 普通数字 |
| 现有资产 | 当前资金 | 高亮强调色 |
| 总盈亏 | 累计盈亏金额 | 红/绿着色 |

| 盈利次数 | 盈利交易次数 | 绿色 |
| 成功率 | 胜率百分比 | 绿色 |

| 最大亏损 | 单笔最大亏损 | 红色 |
| 最大回撤 | 最大回撤比例 | 普通数字 |

#### 3.2.2 选股条件

**条件类型（双列布局）：**

| 分类 | 条件选项 |
|------|----------|
| 趋势向上 | 随机、历史新高、1年新高、200日新高、30日涨幅前50、15日涨幅前50、涨停、连板、量升价涨、上升趋势 |
| 趋势向下 | 历史新低、1年新低、200日新低、30日跌幅前50、15日跌幅前50、下降趋势、跌停、连续跌停 |

**交互说明：**
- 单选模式，默认选中"随机"
- 选中后文字变为强调色
- 底部显示满足条件的股票数量

#### 3.2.3 最近交易列表

| 字段 | 说明 |
|------|------|
| 股票名称 | 如：宁德时代 |
| 交易类型 | 买入/卖出 + 数量 |
| 盈亏金额 | 红/绿显示 |
| 盈亏比例 | 百分比 |
| 状态标签 | 持仓(绿)/止损(红)/已平仓(灰) |

### 4.2 功能组件

#### 4.2.1 股票信息区

| 字段 | 说明 | 颜色规则 |
|------|------|----------|
| 股票名称 | 如：宁德时代 | 黑色 |
| 股票代码 | 如：SZ 300750 | 灰色 |

#### 4.2.2 行情数据网格

| 字段 | 说明 |
|------|------|
| 今开 | 今日开盘价 |
| 最高 | 今日最高价 |
| 最低 | 今日最低价 |

| 成交额 | 当日成交额 |

| 市盈率 | PE |
| 市净率 | PB |
| 60日高 | 60日最高价 |
| 60日低 | 60日最低价 |
| 涨停价 | 涨停价格 |

#### 4.2.3 K线图区域

**主图元素：**
- K线蜡烛图（红涨绿跌）
- MA5/MA10/MA20/MA30 四条均线
- 买卖点标记（圆圈标记）
- 网格线

**副图指标：**
- 成交量（柱状图）
- MACD
- KDJ
- RSI
- BOLL

**周期切换：**
- 日K / 周K / 月K / 季K / 年K
- 选中状态：强调色背景 + 白色文字

#### 4.2.4 交易操作区

**交易按钮：**

| 按钮 | 功能 | 样式 |
|------|------|------|
| 换股 | 切换训练标的 | 白底灰边 |
| 条件单 | 设置止盈止损/网格 | 白底灰边 |
| 买入 | 执行买入操作 | 绿色实心 |
| 卖出 | 执行卖出操作 | 红色实心 |

**交易信息面板：**

| 字段 | 说明 |
|------|------|
| 持仓/可用 | 当前持仓数量 / 可卖数量 |
| 成本/现价 | 持仓成本价 / 当前市场价 |
| 市值/盈亏 | 持仓市值 / 盈亏金额（带badge） |
| 总资产/可用 | 总资产 / 可用资金 |
| 今日盈亏 | 当日盈亏金额和比例 |
| 总盈亏 | 累计盈亏金额和比例 |

#### 4.2.5 条件单系统

**条件单类型：**

| 类型 | 触发条件 | 适用场景 |
|------|----------|----------|
| 止盈单 | 价格 >= 设定止盈价 | 锁定利润 |
| 止损单 | 价格 <= 设定止损价 | 控制亏损 |
| 条件买入 | 价格 >= 或 <= 设定价 | 突破/回调买入 |
| 条件卖出 | 价格 >= 或 <= 设定价 | 突破/反弹卖出 |
| 网格交易 | 在价格区间内自动低买高卖 | 震荡市获利 |

**网格交易参数：**
- 上限价格
- 下限价格
- 网格数量
- 每格投入金额
