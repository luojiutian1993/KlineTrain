## 预防措施

### 架构违规预防

### 命名不规范预防

- [ ] 写代码前参考命名约定表
- [ ] 使用IDE的代码模板
- [ ] 代码审查时重点检查命名

### 逻辑错误预防

- [ ] 编写单元测试
- [ ] 复杂逻辑添加注释
- [ ] 参考现有实现模式

### 性能问题预防

- [ ] 使用const构造函数
