# K线训练营 - 需求总览文档

## 1. 项目概述

### 1.1 项目愿景
> 成为国内领先的K线模拟训练平台，帮助初级交易员通过历史真实K线训练提升盘感，降低真实交易的学习成本和风险。

### 1.2 核心价值
- **降低学习门槛**：无需真金白银即可进行真实市场环境的模拟训练
- **提升交易技能**：通过反复练习K线形态识别，提高对短期走势的判断能力
- **建立交易纪律**：通过条件单（止盈止损、网格交易）训练良好的风控习惯
- **数据驱动改进**：基于训练记录统计，清晰了解自身交易风格和短板
- **沉浸式体验**：平滑的K线走势和精准的买卖点位标记，提升训练真实感

### 1.3 目标用户
- **[股票/期货新手]**：刚进入市场，缺乏实战经验，希望先模拟练习再入场
- **[有亏损经历的初级交易员]**：有一定交易经验但胜率不高，希望提升盘感
- **[自我驱动的学习者]**：希望通过系统化训练形成自己的交易体系

### 5.1 CCI指标 [REQ-BATTLE-025]
**功能描述**：顺势指标(Commodity Channel Index)，用于判断市场的超买超卖状态

**算法**：
```
TP = (High + Low + Close) / 3
SMA_TP = TP的N日简单移动平均
MD = TP与其N日SMA的绝对差的N日平均
CCI = (TP - SMA_TP) / (0.015 × MD)
```

**参数**：N = 14

**验收标准**：
- [ ] CCI数值计算正确
- [ ] 图表正确渲染CCI曲线
- [ ] 超买超卖区域正确标识（>100超买，<-100超卖）

### 5.2 OBV指标 [REQ-BATTLE-026]
**功能描述**：能量潮(On Balance Volume)，用于判断成交量与价格走势的关系

**算法**：
```
如果今日收盘价 > 昨日收盘价：OBV = OBV昨日 + 今日成交量
如果今日收盘价 < 昨日收盘价：OBV = OBV昨日 - 今日成交量
如果今日收盘价 = 昨日收盘价：OBV = OBV昨日
```

**验收标准**：
- [ ] OBV数值计算正确
- [ ] 图表正确渲染OBV曲线
- [ ] 与K线走势联动显示

### 5.3 DMI指标 [REQ-BATTLE-027]
**功能描述**：动向指标(Directional Movement Index)，包括+DI、-DI和ADX

**算法**：
```
+DM = 今日最高价 - 昨日最高价（仅当>0且>|-DM|时取正值）
-DM = 昨日最低价 - 今日最低价（仅当>0且>|+DM|时取正值）
TR = max(最高-最低, |最高-昨日收盘|, |最低-昨日收盘|)
+DI = (+DM的EMA / TR) × 100
-DI = (-DM的EMA / TR) × 100
DX = (|+DI - -DI| / |+DI + -DI|) × 100
ADX = DX的EMA
```

**参数**：N = 14

**验收标准**：
- [ ] +DI、-DI、ADX数值计算正确
- [ ] 图表正确渲染三条曲线
- [ ] 金叉死叉信号正确标识

### 5.4 DMA指标 [REQ-BATTLE-028]
**功能描述**：差分平均线(Different Moving Average)，快线与慢线的差值

**算法**：
```
快线 = 收盘价的短周期SMA
慢线 = 收盘价的长周期SMA
DMA = 快线 - 慢线
AMA = DMA的EMA
```

**参数**：短周期 = 10，长周期 = 50，signal = 10

**验收标准**：
- [ ] DMA和AMA数值计算正确
- [ ] 图表正确渲染DMA和AMA曲线
- [ ] 金叉死叉信号正确标识

### 5.5 BBI指标 [REQ-BATTLE-029]

**算法**：
