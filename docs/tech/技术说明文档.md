### 3.3 平滑均线实现
**核心技术**：二阶贝塞尔曲线

```dart
void _drawSmoothLine(Canvas canvas, Size size, List<double> values, Color color) {
  final path = Path();
  bool started = false;

  for (int i = 0; i < values.length && i < klineData.length; i++) {
    if (values[i] == 0 || values[i].isNaN) continue;

    final x = i * candleWidth + candleWidth / 2;
    final y = size.height - ((values[i] - minY) / priceRange) * size.height;

    if (!started) {
      path.moveTo(x, y);
      started = true;
    } else {
      final prevX = (i - 1) * candleWidth + candleWidth / 2;
      final prevY = size.height - ((values[i - 1] - minY) / priceRange) * size.height;
      final cpX = (prevX + x) / 2;
      // 使用二阶贝塞尔曲线平滑过渡
      path.quadraticBezierTo(prevX, prevY, cpX, (prevY + y) / 2);
    }
  }

  canvas.drawPath(path, linePaint);
}
```

**关键点**：
- `Path.quadraticBezierTo()` 实现平滑曲线
- 控制点为前一点和当前点的中点
- `isAntiAlias = true` 抗锯齿

### 3.4 买卖点位标记
**数据结构**：
```dart
class TradePoint {
  final int index;       // K线索引
  final double price;    // 交易价格
  final bool isBuy;      // true=买入，false=卖出
  final String label;    // 显示标签
  final DateTime date;   // 交易日期
}
```

**绘制实现**：
```dart
void _drawTradePoints(Canvas canvas, Size size) {
  for (var point in tradePoints!) {
    final x = point.index * candleWidth + candleWidth / 2;
    final y = size.height - ((point.price - minY) / priceRange) * size.height;

    final paint = Paint()
      ..color = point.isBuy ? Colors.red : Colors.green
      ..style = PaintingStyle.fill
      ..isAntiAlias = true;

    final path = Path();
    const triangleSize = 8.0;

    if (point.isBuy) {
      // 向上三角形
      path.moveTo(x, y - triangleSize);
      path.lineTo(x - triangleSize, y);
      path.lineTo(x + triangleSize, y);
    } else {
      // 向下三角形
      path.moveTo(x, y + triangleSize);
      path.lineTo(x - triangleSize, y);
      path.lineTo(x + triangleSize, y);
    }
    path.close();
    canvas.drawPath(path, paint);
  }
}
```

### 3.5 时间步进器实现
**自动播放逻辑**：
```dart
void _togglePlay() {
  setState(() {
    _isPlaying = !_isPlaying;
  });

  if (_isPlaying) {
    _playTimer = Timer.periodic(const Duration(seconds: 2), (timer) {
      if (_currentDayIndex < _allKlineData.length - 1) {
        setState(() {
          _currentDayIndex++;
          _checkConditionalOrders();
          _updateAccount();
        });
      } else {
        setState(() {
          _isPlaying = false;
        });
        timer.cancel();
      }
    });
  } else {
    _playTimer?.cancel();
  }
}
```

    quantity = (maxBuy * ratio).floorToDouble();

## 4. 数据层技术实现

### 4.1 K线仓库
**文件**：`lib/data/repositories/kline_repository.dart`

**数据获取优先级**：
```dart
Future<List<KlineModel>> fetchKlineData({
  String symbol = 'SH600000',
  String timeframe = 'day',
  int limit = 100,
}) async {
  // 1. 优先从数据库读取
  final dbData = await fetchKlineDataFromDb(symbol: symbol, period: timeframe, limit: limit);
  if (dbData.isNotEmpty) {
    return dbData;
  }

  // 2. 数据库没有从API获取
  final apiData = await _api.fetchKlineData(symbol: symbol, timeframe: timeframe, limit: limit);
  if (apiData.isNotEmpty) {
    await _saveToDatabase(apiData, timeframe);
    return apiData;
  }

  // 3. 兜底使用模拟数据
  return _generateMockData(symbol, limit);
}
```

**数据库存储**：
```dart
Future<void> _saveToDatabase(List<KlineModel> data, String period) async {
  final dbService = await _getDbService();
  final companions = data.map((item) {
    return KlineDataCompanion.insert(
      symbol: item.symbol,
      marketCode: '',
      period: period,
      tradeDate: DateTime.fromMillisecondsSinceEpoch(item.timestamp),
      open: item.open,
      high: item.high,
      low: item.low,
      close: item.close,
      volume: item.volume,
      amount: item.turnover,
    );
  }).toList();

  await dbService.klineDao.batchInsertKline(companions);
}
```
