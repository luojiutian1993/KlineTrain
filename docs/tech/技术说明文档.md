### 3.3 平滑均线实现
**核心技术**：二阶贝塞尔曲线

```dart
void _drawSmoothLine(Canvas canvas, Size size, List<double> values, Color color) {
  final path = Path();
  bool started = false;

  for (int i = 0; i < values.length && i < klineData.length; i++) {
    if (values[i] == 0 || values[i].isNaN) continue;

    final x = i * candleWidth + candleWidth / 2;
    final y = size.height - ((values[i] - minY) / priceRange) * size.height;

    if (!started) {
      path.moveTo(x, y);
      started = true;
    } else {
      final prevX = (i - 1) * candleWidth + candleWidth / 2;
      final prevY = size.height - ((values[i - 1] - minY) / priceRange) * size.height;
      final cpX = (prevX + x) / 2;
      // 使用二阶贝塞尔曲线平滑过渡
      path.quadraticBezierTo(prevX, prevY, cpX, (prevY + y) / 2);
    }
  }

  canvas.drawPath(path, linePaint);
}
```

**关键点**：
- `Path.quadraticBezierTo()` 实现平滑曲线
- 控制点为前一点和当前点的中点
- `isAntiAlias = true` 抗锯齿

**绘制实现**：
```dart
void _drawTradePoints(Canvas canvas, Size size) {
  for (var point in tradePoints!) {
    final x = point.index * candleWidth + candleWidth / 2;
    final y = size.height - ((point.price - minY) / priceRange) * size.height;

    final paint = Paint()
      ..color = point.isBuy ? Colors.red : Colors.green
      ..style = PaintingStyle.fill
      ..isAntiAlias = true;

    final path = Path();
    const triangleSize = 8.0;

    if (point.isBuy) {
      // 向上三角形
      path.moveTo(x, y - triangleSize);
      path.lineTo(x - triangleSize, y);
      path.lineTo(x + triangleSize, y);
    } else {
      // 向下三角形
      path.moveTo(x, y + triangleSize);
      path.lineTo(x - triangleSize, y);
      path.lineTo(x + triangleSize, y);
    }
    path.close();
    canvas.drawPath(path, paint);
  }
}
```

## 8. 指标计算实现

### 8.1 指标计算工具类
**文件**：`lib/data/utils/indicator_calculator.dart`

**核心方法**：
```dart
class IndicatorCalculator {
  // MACD指标：DIF快线、DEA慢线、MACD柱状图
  static MACDResult calculateMACD(List<KlineModel> data, {
    int fastPeriod = 12,
    int slowPeriod = 26,
    int signalPeriod = 9,
  });

  // KDJ随机指标：K线、D线、J线
  static KDJResult calculateKDJ(List<KlineModel> data, {
    int rsvPeriod = 9,
    int kPeriod = 3,
    int dPeriod = 3,
  });

  // RSI相对强弱指标
  static RSIResult calculateRSI(List<KlineModel> data, {int period = 14});

  // BOLL布林带：上轨、中轨、下轨
  static BollResult calculateBoll(List<KlineModel> data, {
    int period = 20,
    double stdDev = 2.0,
  });

  // WR威廉指标
  static WRResult calculateWR(List<KlineModel> data, {int period = 14});

  // EMA指数移动平均
  static List<double> calculateEMA(List<double> values, int period);

  // SMA简单移动平均
  static List<double> calculateSMA(List<double> values, int period);
}
```

### 8.2 MACD计算详解
```dart
// 1. 计算快速和慢速EMA
EMA12 = calculateEMA(closes, 12);
EMA26 = calculateEMA(closes, 26);

// 2. DIF = EMA12 - EMA26
DIF[i] = EMA12[i] - EMA26[i];

// 3. DEA = EMA(DIF, 9)
DEA = calculateEMA(DIF, 9);

// 4. MACD柱 = 2 * (DIF - DEA)
MACD[i] = 2 * (DIF[i] - DEA[i]);
```

### 8.3 KDJ计算详解
```dart
// 1. 计算RSV
lowestLow = LLV(lows, 9);
highestHigh = HHV(highs, 9);
RSV = (close - lowestLow) / (highestHigh - lowestLow) * 100;

// 2. K、D、J平滑计算
K = 2/3 * K_prev + 1/3 * RSV;
D = 2/3 * D_prev + 1/3 * K;
J = 3 * K - 2 * D;
```

### 9.2 季线聚合示例
```sql
SELECT
  symbol,
  'quarter' as period,
  CASE
    WHEN strftime('%m', trade_date) BETWEEN '01' AND '03'
      THEN date(date_trunc('year', trade_date), '+2 months', 'start of month', '+1 month', '-1 day')
    WHEN strftime('%m', trade_date) BETWEEN '04' AND '06'
      THEN date(date_trunc('year', trade_date), '+5 months', ...)
    -- Q3, Q4 同理
  END as trade_date,
  FIRST_VALUE(open) OVER (PARTITION BY symbol, quarter ORDER BY trade_date) as open,
  MAX(high) OVER (PARTITION BY symbol, quarter) as high,
  MIN(low) OVER (PARTITION BY symbol, quarter) as low,
  LAST_VALUE(close) OVER (PARTITION BY symbol, quarter ORDER BY trade_date) as close,
  SUM(volume) OVER (PARTITION BY symbol, quarter) as volume,
  SUM(amount) OVER (PARTITION BY symbol, quarter) as amount
FROM kline_data
WHERE symbol = ? AND period = 'day'
```

| `_scaleStart` | `double` | 缩放起始点 | 0.0 |
| `_offsetStart` | `double` | 偏移起始点 | 0.0 |

### 11.2 手势处理
**双指缩放**：
```dart
void _handleScaleStart(ScaleStartDetails details) {
  _scaleStart = _currentScale;
  _offsetStart = _offsetX;
}

void _handleScaleUpdate(ScaleUpdateDetails details) {
  double newScale = _scaleStart * details.scale;
  newScale = max(_minScale, min(_maxScale, newScale));
  
  setState(() {
    _currentScale = newScale;
  });
}
```

**单指滑动**：

```dart
}

    _offsetX = max(_minOffset, min(_maxOffset, _offsetX));
  });
}
```

```dart
  });

  });
}
```

**胜率计算**：
```dart
int _calculateWinCount() {
  if (_tradePoints.isEmpty) return 0;
  int winCount = 0;
  bool hasPosition = false;
  double cost = 0;

  for (final point in _tradePoints) {
    if (point.isBuy) {
      hasPosition = true;
      cost = point.price;
    } else if (hasPosition) {
      if (point.price > cost) winCount++;
      hasPosition = false;
    }
  }
  return winCount;
}

double _calculateWinRate() {
  if (_tradePoints.isEmpty) return 0.0;
  final sellCount = _tradePoints.where((p) => !p.isBuy).length;
  if (sellCount == 0) return 0.0;
  return (_calculateWinCount() / sellCount) * 100;
}
```
