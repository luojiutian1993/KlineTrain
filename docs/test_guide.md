## 测试注意事项

1. **数据隔离**: 每次测试前可清除应用数据或使用不同手机号注册
2. **网络状态**: 当前实现为本地数据库操作，无需网络连接
3. **数据持久化**: 退出应用后重新启动，用户数据应仍然存在
4. **重复注册**: 使用相同手机号重复注册可能导致唯一约束冲突（取决于实现）

## 问题排查

### 常见问题及解决方案

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 注册后页面无数据 | 数据库未正确初始化 | 检查DatabaseService.initialize()调用 |
| 昵称更新后不显示 | 状态管理未更新 | 检查UserNotifier.refreshProfile() |
| 注册失败 | 手机号格式错误 | 确保输入11位手机号 |
| 数据库文件找不到 | 应用沙盒路径错误 | 使用Xcode查看应用容器 |
