## 功能模块文档索引

### 1. 选股条件功能

#### 功能概述
实现18种选股条件算法，支持按市场板块和时间范围筛选股票。

#### 核心文档

| 文档类型 | 文件路径 | 说明 |
|----------|----------|------|
| 需求文档 | [选股条件功能需求](../superpowers/requirements/2026-05-16-stock-selection-functional-requirements.md) | 功能需求详细说明 |
| 技术设计 | [选股条件技术设计](../superpowers/designs/2026-05-16-stock-selection-technical-design.md) | 架构设计、算法实现 |
| 开发记录 | [时间范围筛选功能开发记录](../开发记录/时间范围筛选功能开发记录.md) | 时间范围功能开发过程 |
| UI优化 | [选股条件UI优化计划](../superpowers/plans/2026-05-14-stock-condition-ui-optimization.md) | UI设计规划 |

#### 功能特性

**选股条件类型（18种）**

趋势向上类（9种）：
1. 历史新高 (allTimeHigh)
2. 一年新高 (yearHigh)
3. 200日新高 (day200High)
4. 30日涨幅前50% (return30dTop)
5. 15日涨幅前50% (return15dTop)
6. 涨停 (limitUp)
7. 连板 (consecutiveLimitUp)
8. 量价齐升 (volumePriceUp)
9. 上升趋势 (upTrend)

趋势向下类（9种）：
10. 历史新低 (allTimeLow)
11. 一年新低 (yearLow)
12. 200日新低 (day200Low)
13. 30日跌幅前50% (loss30dTop)
14. 15日跌幅前50% (loss15dTop)
15. 下降趋势 (downTrend)
16. 跌停 (limitDown)
17. 连续跌停 (consecutiveLimitDown)
18. 随机 (random)

**筛选维度**
- 市场板块：A 股、港股、美股、期货
- 时间范围：近1年、近3年、近5年、自定义

## 代码结构索引

### 核心枚举
- `lib/core/enums/stock_filter_condition.dart` - 选股条件枚举、筛选方向、时间范围类型

### 数据模型
- `lib/data/models/stock_filter_condition_model.dart` - 选股条件模型
- `lib/data/models/stock_filter_result_model.dart` - 选股结果模型

### 数据库表
- `lib/data/database/tables/stock_filter_results_table.dart` - 选股结果缓存表
- `lib/data/database/tables/daily_stock_stats_table.dart` - 每日股票统计表

### DAO层
- `lib/data/database/daos/stock_filter_dao.dart` - 选股算法实现

### Repository层
- `lib/data/repositories/stock_filter_repository.dart` - 选股业务逻辑

### Provider层
- `lib/providers/stock_filter_provider.dart` - 选股状态管理
- `lib/providers/selection_provider.dart` - 选股与K线数据联动

### UI组件
- `lib/features/home/widgets/market_sector_selector.dart` - 市场板块选择器
- `lib/features/home/widgets/stock_condition_selector.dart` - 选股条件选择器

## 数据库架构

### 选股结果缓存表 (stock_filter_results)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| filter_date | DATETIME | 选股日期 |
| condition_type | TEXT | 选股条件 |
| symbol | TEXT | 标的代码 |
| market_code | TEXT | 市场代码 |
| symbol_name | TEXT | 标的名称 |
| close_price | REAL | 收盘价 |
| change_percent | REAL | 涨跌幅 |

### 每日股票统计表 (daily_stock_stats)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| trade_date | DATETIME | 交易日期 |
| symbol | TEXT | 标的代码 |
| close_price | REAL | 收盘价 |
| return15d | REAL | 15日涨幅 |
| return30d | REAL | 30日涨幅 |
| ma10/ma20/ma50/ma200 | REAL | 均线值 |
