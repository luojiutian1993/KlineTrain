# 实战页面布局优化开发记录

## 一、需求概述

### 需求来源
根据用户反馈，实战页面需要进行以下优化：
1. **单屏幕展示**：所有信息在一个页面中展示，无需上下滑动
2. **布局调整**：缩小字体，删除冗余标签，合并信息行
3. **指标联动**：K线与指标图表同步联动
4. **边缘提醒**：滑动到达边缘时弹出提醒

### 需求编号
- REQ-017: 实战页面单屏幕布局优化
- REQ-018: 指标图表初始化显示完整历史数据
- REQ-019: K线与指标图表联动同步
- REQ-020: 边缘滑动提醒功能

### 2.1 单屏幕布局优化

**修改文件**: `lib/features/battle/battle_screen.dart`

### 2.2 指标图表初始化显示完整历史数据

**问题分析**:
- 初始化时 `_visibleStartIndex` 设置错误，导致显示的是最后部分数据
- MACD等指标计算使用 `_allKlineData.length` 作为边界，包含未训练的数据

**修改内容**:
```dart
// 修改 _loadKlineData() 中 _visibleStartIndex 的初始化
final endIndex = _currentDayIndex + 1;
_visibleStartIndex = (endIndex - _visibleKlineCount).clamp(0, endIndex);

// 修改所有指标计算逻辑使用统一的边界
final endIndex = (_currentDayIndex + 1).clamp(0, _allKlineData.length);
```

**修改的指标**: MACD、KDJ、RSI、BOLL、WR、CCI、OBV、DMI、DMA、BBI

### 2.3 K线与指标图表联动同步

**问题分析**:
- `_nextDay()` 方法没有更新 `_visibleStartIndex`
- 导致K线图表滑动但指标图表数据不同步

**修改内容**:
```dart
void _nextDay() {
  if (_currentDayIndex < _allKlineData.length - 1) {
    setState(() {
      _currentDayIndex++;
      final endIndex = _currentDayIndex + 1;
      final maxStart = (endIndex - _visibleKlineCount).clamp(0, endIndex);
      _visibleStartIndex = maxStart;
      // ...
    });
  }
}
```

### 2.4 边缘滑动提醒功能

**修改内容**:
```dart
void _slideRight() {
  final endIndex = (_currentDayIndex + 1).clamp(0, _allKlineData.length);
  final maxStart = (endIndex - _visibleKlineCount).clamp(0, endIndex);
  if (_visibleStartIndex >= maxStart) {
    _showEdgeAlert('已经到达最右边');
    return;
  }
  // ...
}

void _showEdgeAlert(String message) {
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(
      content: Text(message),
      duration: Duration(seconds: 1),
    ),
  );
}
```
