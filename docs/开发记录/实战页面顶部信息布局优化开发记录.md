## 1. 需求概述

本次开发主要包含三个需求：

1. **边缘提醒防抖** (REQ-021): 用户短时间内多次点击左右按钮到达边缘时，只需提醒一次
2. **顶部股票信息布局重新设计** (REQ-022): 将实战页面顶部信息重新布局为三行结构
3. **高、低等字段垂直对齐** (REQ-023): 确保高、低等字段以及对应数据垂直对齐显示

## 2. 修改内容

### 2.1 边缘提醒防抖

**修改文件**: `lib/features/battle/battle_screen.dart`

**变更点**:
1. 在 `_BattleScreenState` 类中添加了 `_lastEdgeAlertTime` 状态变量
2. 修改 `_showEdgeAlert()` 方法，添加时间间隔检查逻辑

**代码变更**:
```dart
// 新增状态变量
DateTime? _lastEdgeAlertTime;

// 修改后的 _showEdgeAlert 方法
void _showEdgeAlert(String message) {
  final now = DateTime.now();
  if (_lastEdgeAlertTime != null) {
    final diff = now.difference(_lastEdgeAlertTime!);
    if (diff.inMilliseconds < 1500) {
      return; // 1500ms内重复点击不提醒
    }
  }
  _lastEdgeAlertTime = now;
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(
      content: Text(message),
      duration: const Duration(milliseconds: 1500),
      behavior: SnackBarBehavior.floating,
    ),
  );
}
```

### 2.3 高、低等字段垂直对齐

**修改文件**: `lib/features/battle/battle_screen.dart`

**布局结构**:
使用 5 个 Column 分别包裹垂直对齐的字段对：

| 列序号 | 第一行 | 第二行 |
|-------|--------|--------|
| 1 | 收盘价 + 收字 | 价差 + 涨跌幅 |
| 2 | 高 | 低 |
| 3 | 开 | 换手 |
| 4 | 流通 | 上证 |
| 5 | 量 | 金额 |

**代码变更**:
```dart
Row(
  crossAxisAlignment: CrossAxisAlignment.start,
  children: [
    // 第一列: 收盘价/涨跌
    Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        Row(
          children: [
            Text(currentData?.close.toStringAsFixed(2) ?? '--',
                style: const TextStyle(...)),
            const SizedBox(width: 4),
            Text('收', style: TextStyle(...)),
          ],
        ),
        const SizedBox(height: 4),
        Row(
          children: [
            Text(..., style: TextStyle(...)),
            const SizedBox(width: 4),
            Text(..., style: TextStyle(...)),
          ],
        ),
      ],
    ),
    const SizedBox(width: 12),
    // 第二列: 高/低
    Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        _buildInfoItem('高', currentData?.high.toStringAsFixed(2) ?? '--'),
        const SizedBox(height: 4),
        _buildInfoItem('低', currentData?.low.toStringAsFixed(2) ?? '--'),
      ],
    ),
    // ... 其他列类似
  ],
)
```

**辅助方法** (`_buildInfoItem`):
```dart
Widget _buildInfoItem(String label, String value) {
  return Row(
    children: [
      Text(label, style: TextStyle(fontSize: 10, color: AppTheme.muted)),
      const SizedBox(width: 4),
      Text(value, style: const TextStyle(fontSize: 11)),
    ],
  );
}
```
