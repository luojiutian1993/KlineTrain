## 需求背景

### 本次对话中的需求

1. **REQ-HOME-009**: 选股条件筛选出符合条件的股票，需要展示具体股票信息
2. **REQ-HOME-010**: 将筛选出符合条件的股票选择某一只具体股票后，点击确认，跳转到实战页面
3. **REQ-HOME-011**: 实战的股票就是根据训练周期时间和选股条件选出的股票，进行实战训练

5. **REQ-HOME-013**: 市场代码映射问题（前端XSHG/XSHE vs 数据库SH/SZ）

## 问题分析

### 问题1：选股结果为0

**现象**: 
- 数据库中有数据，但选股结果始终为0
- 测试多种选股条件都没有结果

**根本原因**:
1. **数据库中没有数据**: 之前没有初始化示例数据
2. **市场代码不匹配**: 前端使用XSHG/XSHE，数据库中是SH/SZ
3. **没有传递市场代码参数**: `_executeStockFilter()`方法中没有传递marketCode
4. **选股日期错误**: 使用当前日期，但数据库中没有对应日期的K线数据

### 问题2：时间范围选择没有反应

**现象**:
- 点击"近1年"可以显示
- 点击"近3年"、"近5年"没有反应

**根本原因**:
- `HomeScreen`调用`StockConditionSelector`时没有传递`selectedTimeRange`和`onTimeRangeChanged`参数

### 问题3：数据库路径问题

**用户疑问**:
- 为什么数据库不在项目路径下
- 数据库文件在哪里

**答案**:
- iOS/macOS应用使用沙箱机制
- 数据必须存储在特定目录
- 沙箱路径: `~/Library/Developer/CoreSimulator/Devices/.../Documents/kline_trainer.db`

```dart
String _mapMarketCode(String marketCode) {
  switch (marketCode) {
    case 'XSHG': return 'SH';
    case 'XSHE': return 'SZ';
    case 'HKEX': return 'HK';
    case 'NASDAQ':
    case 'NYSE': return 'US';
    default: return marketCode;
  }
}

Future<List<SymbolData>> getActiveSymbols({String? marketCode}) async {
  final query = select(symbols)
    ..where((t) => t.enabled.equals(true));
  
  if (marketCode != null) {
    final mappedCode = _mapMarketCode(marketCode);
    query.where((t) => t.marketCode.equals(mappedCode));
  }
  
  return query.get();
}
```

### 3. 添加获取有K线数据的最近日期

**文件**: `lib/data/database/daos/stock_filter_dao.dart`

```dart
Future<DateTime?> getLatestKlineDate() async {
  final result = await (selectOnly(klineData)
        ..addColumns([klineData.tradeDate.max()])
        ..where(klineData.period.equals('day')))
      .getSingle();
  
  final timestamp = result.read(klineData.tradeDate.max());
  if (timestamp == null) return null;
  
  return DateTime.fromMillisecondsSinceEpoch(timestamp * 1000);
}
```
