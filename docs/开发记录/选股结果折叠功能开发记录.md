# 选股结果列表折叠功能开发记录

## 开发日期
2026-05-25

## 功能需求
首页筛选股票优化：根据选股条件选出股票时，无需全部展示，可只展示3只符合条件的，剩下的用下拉隐藏，如果需要查看点击下拉后展示

## 实现方案

### 修改文件
- `lib/features/home/widgets/stock_condition_selector.dart`

### 修改内容

#### 1. 添加状态变量
在 `_StockConditionSelectorState` 类中添加：
- `_isExpanded`：控制展开/折叠状态
- `_maxVisibleCount`：最大显示数量（固定为3）

```dart
bool _isExpanded = false;
static const int _maxVisibleCount = 3;
```

#### 2. 修改 `_buildStockList` 方法
- 显示总数：在标题处显示符合条件的股票总数 `选股结果 ($totalCount)`
- 默认显示3条：当结果数量超过3条时，只显示前3条
- 添加展开/收起按钮：当结果数量 > 3 时，显示"更多"按钮
- 点击展开/收起：通过 setState 切换 `_isExpanded` 状态

### 核心逻辑
```dart
final totalCount = result.items.length;
final visibleItems = _isExpanded
    ? result.items
    : result.items.take(_maxVisibleCount).toList();
final hasMore = totalCount > _maxVisibleCount;
```

### UI设计
- 标题行：左侧显示"选股结果 (总数)"，右侧显示"更多/收起"按钮
- 展开按钮：蓝色文字 + 下拉箭头图标
- 收起按钮：蓝色文字 + 上拉箭头图标

## 验收标准
| 验收项 | 标准 |
|--------|------|
| 默认显示 | 结果超过3条时默认显示前3条 |
| 展开功能 | 点击"更多"后显示全部结果 |
| 收起功能 | 点击"收起"后只显示前3条 |
| 总数显示 | 标题正确显示符合条件的股票总数 |
| 按钮隐藏 | 结果≤3条时不显示展开/收起按钮 |
