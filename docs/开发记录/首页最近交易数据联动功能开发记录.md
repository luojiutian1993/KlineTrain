# 首页最近交易数据联动功能开发记录

## 概述

本文档记录首页"最近交易"页面从数据库获取交易记录的实现过程，实现前后端及数据库的联动。

## 开发时间

- 开发日期：2026-05-16
- 功能版本：v1.3 → v1.4

## 需求来源

用户需求：最近交易内容与训练记录一样，先展示股票整体情况，再点击进入包含每笔交易记录。

## 实现概述

### 需求变更说明

将原来的单笔交易列表改为按股票分组的汇总展示：
- **原方案**：显示最近10笔交易记录
- **新方案**：显示最近交易的股票汇总，点击进入查看每笔交易

### 新增文件

#### 1. StockTradeSummaryModel（股票交易汇总模型）
位置：`lib/data/models/stock_trade_summary_model.dart`

```dart
class StockTradeSummaryModel {
  final String symbol;
  final String symbolName;
  final String marketCode;
  final int sessionCount;       // 会话数量
  final int totalTradeCount;    // 总交易笔数
  final double totalProfit;    // 总盈亏
  final double profitRate;     // 平均收益率
  final int winCount;          // 盈利次数
  final double winRate;         // 胜率
  final DateTime? lastTradeDate; // 最后交易时间

  // 辅助方法
  bool get isWin => totalProfit > 0;
  String get displayProfit => '+¥XXX' / '-¥XXX';
  String get displayProfitRate => '+X.XX%' / '-X.XX%';
  String get displayWinRate => 'XX%';
  String get displayTradeCount => 'X笔';
}
```

#### 2. StockTradeSummaryProvider（股票交易汇总状态管理）
位置：`lib/providers/stock_trade_summary_provider.dart`

```dart
final stockTradeSummaryProvider =
    AsyncNotifierProvider<StockTradeSummaryNotifier, List<StockTradeSummaryModel>>(
  StockTradeSummaryNotifier.new,
);

// 实现逻辑：
// 1. 获取最近100条交易记录
// 2. 按股票代码分组 (marketCode + symbol)
// 3. 聚合计算每个股票的统计数据
// 4. 按最后交易时间排序
// 5. 返回最近10只股票
```

### 修改文件

#### 1. app_routes.dart（添加路由）
- 新增 `/trade-detail` 路由
- 支持 query parameters: symbol, name, market

#### 2. HomeScreen（更新展示方式）
- 使用 `stockTradeSummaryProvider` 替代 `recentTradesProvider`
- 新增 `_buildStockTradeSummaryContent()` 显示股票汇总卡片
- 每条记录可点击跳转到交易详情页面

## 数据流向

```
┌─────────────────────────────────────────────────────────────┐
│                    HomeScreen                                │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  ref.watch(stockTradeSummaryProvider)                  │   │
│  │    ↓                                                 │   │
│  │  AsyncValue<List<StockTradeSummaryModel>>              │   │
│  └──────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  StockTradeSummaryNotifier.build()                    │   │
│  │    1. getRecentTradesWithSymbol(limit: 100)           │   │
│  │    2. 按股票分组 (marketCode + symbol)                │   │
│  │    3. 聚合统计数据                                    │   │
│  │    4. 按最后交易时间排序                              │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                           ↓
              ┌────────────────────────────────┐
              │  点击股票卡片 → 跳转详情页面     │
              └────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│                   TradeDetailScreen                         │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  _loadTrades()                                      │   │
│  │    1. 筛选该股票的所有交易记录                       │   │
│  │    2. 按时间排序                                    │   │
│  │    3. 计算汇总统计                                  │   │
│  └──────────────────────────────────────────────────────┘   │
│                           ↓                                  │
│  显示：总盈亏、平均收益率、交易笔数、胜率 + 交易明细列表    │
└─────────────────────────────────────────────────────────────┘
```

## 测试验证

### 测试场景

1. **首次启动（无交易数据）**
   - 显示空状态："暂无交易记录"

2. **有交易数据后**
   - 显示股票汇总列表（按最后交易时间排序）
   - 每只股票显示：名称、代码、收益率、盈亏、交易笔数、胜率

3. **点击股票卡片**
   - 跳转到交易详情页面
   - 显示该股票的所有交易明细

4. **加载状态**
   - 显示加载动画
   - 加载完成后显示数据
