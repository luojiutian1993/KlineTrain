# 首页资产卡片样式修改记录

## 概述

本文档记录首页资产统计卡片样式的一致性调整。

## 修改日期

- 修改日期：2026-05-16
- 功能版本：v1.4.2

## 需求来源

用户需求：
1. 现有资产的字体格式与初始资产保持一致
2. 收益率不显示其他底色，与模块底色保持一致

## 修改内容

### HomeScreen._buildAssetCardContent()

位置：`lib/features/home/home_screen.dart`

**修改前**：
```dart
Row(
  children: [
    Expanded(
      child: _StatItem(
        label: '初始资产',
        value: _formatCurrency(summary.initialCapital),
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '现有资产',
        value: _formatCurrency(summary.currentCapital),
        highlight: true,  // 高亮显示
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '总盈亏',
        value: '$profitPrefix${_formatCurrency(summary.totalProfit)}',
        color: profitColor,
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '收益率',
        value: _formatPercent(summary.profitRate),
        bgHighlight: true,  // 背景高亮
      ),
    ),
  ],
),
```

**修改后**：
```dart
Row(
  children: [
    Expanded(
      child: _StatItem(
        label: '初始资产',
        value: _formatCurrency(summary.initialCapital),
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '现有资产',
        value: _formatCurrency(summary.currentCapital),
        // 移除highlight样式，与初始资产保持一致
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '总盈亏',
        value: '$profitPrefix${_formatCurrency(summary.totalProfit)}',
        color: profitColor,
      ),
    ),
    Expanded(
      child: _StatItem(
        label: '收益率',
        value: _formatPercent(summary.profitRate),
        // 移除bgHighlight样式，使用默认底色
      ),
    ),
  ],
),
```

## 修改效果

### 修改前样式
| 字段 | 样式 |
|------|------|
| 初始资产 | 默认格式 |
| 现有资产 | 高亮+粗体 |
| 总盈亏 | 盈亏颜色（红/绿） |
| 收益率 | 背景高亮+粗体 |

### 修改后样式
| 字段 | 样式 |
|------|------|
| 初始资产 | 默认格式 |
| 现有资产 | 默认格式（与初始资产一致） |
| 总盈亏 | 盈亏颜色（红/绿） |
| 收益率 | 默认格式（无特殊背景） |

## 文件变更清单

| 文件路径 | 修改内容 |
|----------|----------|
| `lib/features/home/home_screen.dart` | 移除现有资产和收益率的高亮样式 |
