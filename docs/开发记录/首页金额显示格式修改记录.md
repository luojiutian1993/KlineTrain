# 首页金额显示格式修改记录

## 概述

本文档记录首页训练模块金额显示格式的修改，将涉及金额的字段从保留0位小数修改为保留2位小数。

## 修改日期

- 修改日期：2026-05-16
- 功能版本：v1.4.1

## 需求来源

用户需求：首页的训练模块，初始资产、现有资产等涉及到金额的需要以元为单位保留两位小数，例如10000.00

## 修改内容

### 1. HomeScreen

位置：`lib/features/home/home_screen.dart`

**修改前**：
```dart
String _formatCurrency(double value) {
  if (value.abs() >= 10000) {
    return '¥${(value / 10000).toStringAsFixed(1)}万';
  }
  return '¥${value.toStringAsFixed(0)}';
}
```

**修改后**：
```dart
String _formatCurrency(double value) {
  return '¥${value.toStringAsFixed(2)}';
}
```

### 2. RecentTradeModel

位置：`lib/data/models/recent_trade_model.dart`

**修改前**：
```dart
String get displayProfit => profit >= 0
    ? '+¥${profit.toStringAsFixed(0)}'
    : '-¥${profit.abs().toStringAsFixed(0)}';
```

**修改后**：
```dart
String get displayProfit => profit >= 0
    ? '+¥${profit.toStringAsFixed(2)}'
    : '-¥${profit.abs().toStringAsFixed(2)}';
```

### 3. StockTradeSummaryModel

位置：`lib/data/models/stock_trade_summary_model.dart`

**修改前**：
```dart
String get displayProfit => totalProfit >= 0
    ? '+¥${totalProfit.toStringAsFixed(0)}'
    : '-¥${totalProfit.abs().toStringAsFixed(0)}';
```

**修改后**：
```dart
String get displayProfit => totalProfit >= 0
    ? '+¥${totalProfit.toStringAsFixed(2)}'
    : '-¥${totalProfit.abs().toStringAsFixed(2)}';
```

### 4. TradeDetailScreen

位置：`lib/features/home/trade_detail_screen.dart`

**修改内容**：
- 总盈亏显示：`'+¥${_totalProfit.toStringAsFixed(2)}'`
- 金额显示：`'金额: ¥${trade.amount.toStringAsFixed(2)}'`

## 显示效果

### 修改前
| 字段 | 显示 |
|------|------|
| 初始资产 | ¥100,000 |
| 现有资产 | ¥128,450 |
| 总盈亏 | +¥28,450 |

### 修改后
| 字段 | 显示 |
|------|------|
| 初始资产 | ¥100000.00 |
| 现有资产 | ¥128450.00 |
| 总盈亏 | +¥28450.00 |

   - 初始资产应显示为：¥100000.00

2. **收益率显示**（保持原有格式）
   - 收益率：+28.45%（保持1位小数）
