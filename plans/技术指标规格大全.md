## 1. 概述

本文档定义K线训练营系统支持的11种技术指标，包括指标含义、计算公式、参数配置和技术实现。

## 3. 指标详解

### 3.1 VOLUME（成交量）

#### 3.1.1 指标含义

成交量是衡量市场交易活跃度的基础指标，反映一定时间内买卖双方成交的股票数量。

**核心作用**：
- **量价配合**：价涨量增为健康上涨，价涨量缩可能见顶
- **趋势确认**：放量突破更可靠，缩量突破可能假突破
- **反转信号**：底部放量可能见底，顶部放量可能见顶

#### 3.1.2 计算公式

```
成交量 = 当日成交股票数量（股）
成交额 = 当日成交金额（元）

换手率 = 成交量 / 流通股本 × 100%
```

#### 3.1.3 显示方式

```
柱状图显示：
- 上涨日（close ≥ open）：红色柱
- 下跌日（close < open）：绿色柱

可选叠加：
- 5日均量线（MA5）
- 10日均量线（MA10）
```

#### 3.1.4 Flutter实现

```dart
class VolumeIndicator {
  List<VolumeData> calculate(List<KlineData> data) {
    return data.map((d) => VolumeData(
      timestamp: d.timestamp,
      volume: d.volume,
      amount: d.amount,
      isUp: d.close >= d.open,  // 涨跌判断
    )).toList();
  }
  
  // 均量线
  List<double> calculateMA(List<KlineData> data, int period) {
    final ma = <double>[];
    for (int i = period - 1; i < data.length; i++) {
      double sum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        sum += data[j].volume;
      }
      ma.add(sum / period);
    }
    return ma;
  }
}
```

### 3.2 MACD（指数平滑异同移动平均线）

#### 3.2.1 指标含义

MACD（Moving Average Convergence Divergence）由Gerald Appel发明，是判断趋势方向和强度的经典指标。

**核心作用**：
- **DIF上穿DEA**：金叉，买入信号
- **DIF下穿DEA**：死叉，卖出信号
- **MACD柱状图**：红色表示多头力量，绿色表示空头力量
- **背离**：价格创新高但MACD不创新高，可能反转

#### 3.2.2 计算公式

```
EMA(N) = close × K + EMA(t-1) × (1-K)
K = 2 / (N + 1)

DIF = EMA(12) - EMA(26)      // 快线
DEA = EMA(DIF, 9)            // 慢线（信号线）
MACD = 2 × (DIF - DEA)       // 柱状图
```

#### 3.2.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 快线周期 | 12 | 5-50 | 短期EMA周期 |
| 慢线周期 | 26 | 10-100 | 长期EMA周期 |
| 信号线周期 | 9 | 5-30 | DEA计算周期 |

#### 3.2.4 Flutter实现

```dart
class MACDIndicator {
  final int fastPeriod;    // 默认12
  final int slowPeriod;    // 默认26
  final int signalPeriod;  // 默认9
  
  MACDIndicator({
    this.fastPeriod = 12,
    this.slowPeriod = 26,
    this.signalPeriod = 9,
  });
  
  List<double> calculateEMA(List<double> data, int period) {
    if (data.length < period) return [];
    
    final k = 2 / (period + 1);
    final ema = <double>[];
    
    // 初始值取SMA
    double sum = 0;
    for (int i = 0; i < period; i++) {
      sum += data[i];
    }
    ema.add(sum / period);
    
    // 后续按EMA公式计算
    for (int i = period; i < data.length; i++) {
      ema.add(data[i] * k + ema.last * (1 - k));
    }
    
    return ema;
  }
  
  MACDResult calculate(List<KlineData> data) {
    final closes = data.map((d) => d.close).toList();
    
    final emaFast = calculateEMA(closes, fastPeriod);
    final emaSlow = calculateEMA(closes, slowPeriod);
    
    // 对齐数据长度
    final offset = slowPeriod - fastPeriod;
    
    // 计算DIF
    final dif = <double>[];
    for (int i = 0; i < emaSlow.length; i++) {
      dif.add(emaFast[i + offset] - emaSlow[i]);
    }
    
    // 计算DEA
    final dea = calculateEMA(dif, signalPeriod);
    
    // 计算MACD柱状图
    final macd = <double>[];
    for (int i = signalPeriod - 1; i < dea.length; i++) {
      macd.add(2 * (dif[i + signalPeriod - 1] - dea[i]));
    }
    
    return MACDResult(dif: dif, dea: dea, macd: macd);
  }
}

class MACDResult {
  final List<double> dif;   // 快线
  final List<double> dea;   // 慢线
  final List<double> macd;  // 柱状图
  
  MACDResult({required this.dif, required this.dea, required this.macd});
}
```

### 3.3 KDJ（随机指标）

#### 3.3.1 指标含义

KDJ由George Lane发明，通过比较收盘价与价格区间的关系来判断超买超卖。

**核心作用**：
- **K > 80, D > 80**：超买区域，可能回调
- **K < 20, D < 20**：超卖区域，可能反弹
- **金叉/死叉**：K线上穿/下穿D线
- **J值**：J > 100 或 J < 0 为极端区域

#### 3.3.2 计算公式

```
RSV = (close - LLV(low, N)) / (HHV(high, N) - LLV(low, N)) × 100

K = 2/3 × K(t-1) + 1/3 × RSV    // K值
D = 2/3 × D(t-1) + 1/3 × K      // D值
J = 3 × K - 2 × D               // J值

其中：
LLV(low, N) = N日最低价
HHV(high, N) = N日最高价
```

#### 3.3.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| RSV周期 | 9 | 5-30 | 计算RSV的周期 |
| K平滑 | 3 | 2-10 | K值平滑系数 |
| D平滑 | 3 | 2-10 | D值平滑系数 |

#### 3.3.4 Flutter实现

```dart
class KDJIndicator {
  final int n;  // RSV周期，默认9
  final int m1; // K平滑，默认3
  final int m2; // D平滑，默认3
  
  KDJIndicator({this.n = 9, this.m1 = 3, this.m2 = 3});
  
  KDJResult calculate(List<KlineData> data) {
    final k = <double>[];
    final d = <double>[];
    final j = <double>[];
    
    double prevK = 50;  // 初始K值
    double prevD = 50;  // 初始D值
    
    for (int i = n - 1; i < data.length; i++) {
      // 计算N日最高价和最低价
      double highestHigh = double.negativeInfinity;
      double lowestLow = double.infinity;
      
      for (int j = i - n + 1; j <= i; j++) {
        highestHigh = max(highestHigh, data[j].high);
        lowestLow = min(lowestLow, data[j].low);
      }
      
      // 计算RSV
      final range = highestHigh - lowestLow;
      final rsv = range == 0 ? 50 : (data[i].close - lowestLow) / range * 100;
      
      // 计算K、D、J
      final currentK = (m1 - 1) / m1 * prevK + 1 / m1 * rsv;
      final currentD = (m2 - 1) / m2 * prevD + 1 / m2 * currentK;
      final currentJ = 3 * currentK - 2 * currentD;
      
      k.add(currentK);
      d.add(currentD);
      j.add(currentJ);
      
      prevK = currentK;
      prevD = currentD;
    }
    
    return KDJResult(k: k, d: d, j: j);
  }
}

class KDJResult {
  final List<double> k;
  final List<double> d;
  final List<double> j;
  
  KDJResult({required this.k, required this.d, required this.j});
}
```

### 3.4 RSI（相对强弱指标）

#### 3.4.1 指标含义

RSI（Relative Strength Index）由Welles Wilder发明，通过比较一段时间内上涨和下跌的幅度来判断买卖力量。

**核心作用**：
- **RSI > 70**：超买区域，可能回调
- **RSI < 30**：超卖区域，可能反弹
- **背离**：价格与RSI走势不一致，可能反转
- **中轴线50**：RSI > 50为多头市场，RSI < 50为空头市场

#### 3.4.2 计算公式

```
RS = SMA(max(close - close(t-1), 0), N) / SMA(abs(close - close(t-1)), N)

RSI = 100 - 100 / (1 + RS)

即：RSI = 100 × SMA(上涨幅度) / (SMA(上涨幅度) + SMA(下跌幅度))
```

#### 3.4.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期 | 14 | 6-30 | 计算周期 |

#### 3.4.4 Flutter实现

```dart
class RSIIndicator {
  final int period;  // 默认14
  
  RSIIndicator({this.period = 14});
  
  RSIResult calculate(List<KlineData> data) {
    if (data.length < period + 1) return RSIResult(values: []);
    
    final rsi = <double>[];
    
    // 计算价格变化
    final changes = <double>[];
    for (int i = 1; i < data.length; i++) {
      changes.add(data[i].close - data[i - 1].close);
    }
    
    // 计算RSI
    for (int i = period; i < changes.length; i++) {
      double gainSum = 0;
      double lossSum = 0;
      
      for (int j = i - period; j < i; j++) {
        if (changes[j] > 0) {
          gainSum += changes[j];
        } else {
          lossSum += changes[j].abs();
        }
      }
      
      final avgGain = gainSum / period;
      final avgLoss = lossSum / period;
      
      if (avgLoss == 0) {
        rsi.add(100);
      } else {
        final rs = avgGain / avgLoss;
        rsi.add(100 - 100 / (1 + rs));
      }
    }
    
    return RSIResult(values: rsi);
  }
}
```

### 3.5 BOLL（布林带）

#### 3.5.1 指标含义

布林带（Bollinger Bands）由John Bollinger发明，通过统计方法计算价格波动区间。

**核心作用**：
- **上轨**：压力位，价格触及上轨可能回调
- **下轨**：支撑位，价格触及下轨可能反弹
- **带宽**：带宽收窄可能变盘，带宽扩大趋势加强
- **中轨**：趋势方向参考

#### 3.5.2 计算公式

```
中轨(MB) = MA(close, N)
标准差(MD) = SQRT(SUM((close - MB)², N) / N)
上轨(UP) = MB + K × MD
下轨(DN) = MB - K × MD

其中：
N = 计算周期（默认20）
K = 标准差倍数（默认2）
```

#### 3.5.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期 | 20 | 10-60 | 计算周期 |
| 标准差倍数 | 2.0 | 1.0-4.0 | 上下轨距离 |

#### 3.5.4 Flutter实现

```dart
class BOLLIndicator {
  final int period;     // 默认20
  final double stdDev;  // 默认2.0
  
  BOLLIndicator({this.period = 20, this.stdDev = 2.0});
  
  BOLLResult calculate(List<KlineData> data) {
    if (data.length < period) return BOLLResult(mb: [], up: [], dn: []);
    
    final closes = data.map((d) => d.close).toList();
    final mb = <double>[];  // 中轨
    final up = <double>[];  // 上轨
    final dn = <double>[];  // 下轨
    
    for (int i = period - 1; i < closes.length; i++) {
      // 计算MA（中轨）
      double sum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        sum += closes[j];
      }
      final ma = sum / period;
      mb.add(ma);
      
      // 计算标准差
      double variance = 0;
      for (int j = i - period + 1; j <= i; j++) {
        variance += pow(closes[j] - ma, 2);
      }
      final md = sqrt(variance / period);
      
      // 计算上下轨
      up.add(ma + stdDev * md);
      dn.add(ma - stdDev * md);
    }
    
    return BOLLResult(mb: mb, up: up, dn: dn);
  }
}
```

### 3.6 WR（威廉指标）

#### 3.6.1 指标含义

WR（Williams %R）由Larry Williams发明，衡量当前价格在过去一段时间内的相对位置。

**核心作用**：
- **WR > -20**：超买区域，可能回调
- **WR < -80**：超卖区域，可能反弹
- **与RSI互补**：WR关注最高价，RSI关注收盘价

#### 3.6.2 计算公式

```
WR = (HHV(high, N) - close) / (HHV(high, N) - LLV(low, N)) × 100

注意：WR的值通常为负数，范围在0到-100之间
```

#### 3.6.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期 | 14 | 6-30 | 计算周期 |

#### 3.6.4 Flutter实现

```dart
class WRIndicator {
  final int period;  // 默认14
  
  WRIndicator({this.period = 14});
  
  WRResult calculate(List<KlineData> data) {
    if (data.length < period) return WRResult(values: []);
    
    final wr = <double>[];
    
    for (int i = period - 1; i < data.length; i++) {
      double highestHigh = double.negativeInfinity;
      double lowestLow = double.infinity;
      
      for (int j = i - period + 1; j <= i; j++) {
        highestHigh = max(highestHigh, data[j].high);
        lowestLow = min(lowestLow, data[j].low);
      }
      
      final range = highestHigh - lowestLow;
      if (range == 0) {
        wr.add(-50);  // 无波动时取中值
      } else {
        wr.add((highestHigh - data[i].close) / range * -100);
      }
    }
    
    return WRResult(values: wr);
  }
}
```

### 3.7 CCI（顺势指标）

#### 3.7.1 指标含义

CCI（Commodity Channel Index）由Donald Lambert发明，衡量价格偏离统计平均值的程度。

**核心作用**：
- **CCI > +100**：超买区域，多头强势
- **CCI < -100**：超卖区域，空头强势
- **CCI在±100之间**：震荡区域
- **背离**：价格与CCI走势不一致，可能反转

#### 3.7.2 计算公式

```
TP = (high + low + close) / 3    // 典型价格
MA = SMA(TP, N)                   // N日平均
MD = SMA(|TP - MA|, N)           // 平均绝对偏差

CCI = (TP - MA) / (0.015 × MD)

其中：0.015是常数，使大部分CCI值落在±100之间
```

#### 3.7.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期 | 14 | 6-30 | 计算周期 |

#### 3.7.4 Flutter实现

```dart
class CCIIndicator {
  final int period;  // 默认14
  
  CCIIndicator({this.period = 14});
  
  CCIResult calculate(List<KlineData> data) {
    if (data.length < period) return CCIResult(values: []);
    
    final cci = <double>[];
    
    // 计算典型价格
    final tp = data.map((d) => (d.high + d.low + d.close) / 3).toList();
    
    for (int i = period - 1; i < tp.length; i++) {
      // 计算MA
      double sum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        sum += tp[j];
      }
      final ma = sum / period;
      
      // 计算MD（平均绝对偏差）
      double devSum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        devSum += (tp[j] - ma).abs();
      }
      final md = devSum / period;
      
      // 计算CCI
      if (md == 0) {
        cci.add(0);
      } else {
        cci.add((tp[i] - ma) / (0.015 * md));
      }
    }
    
    return CCIResult(values: cci);
  }
}
```

### 3.8 OBV（能量潮）

#### 3.8.1 指标含义

OBV（On Balance Volume）由Joe Granville发明，通过累计成交量来衡量资金流向。

**核心作用**：
- **OBV上升**：资金流入，多头占优
- **OBV下降**：资金流出，空头占优
- **背离**：价格上涨但OBV下降，可能见顶
- **突破**：OBV突破前高，确认趋势

#### 3.8.2 计算公式

```
当日收盘价 > 前日收盘价：OBV = 前日OBV + 当日成交量
当日收盘价 < 前日收盘价：OBV = 前日OBV - 当日成交量
当日收盘价 = 前日收盘价：OBV = 前日OBV

初始OBV = 第一日成交量
```

#### 3.8.3 参数配置

OBV不需要参数配置。

#### 3.8.4 Flutter实现

```dart
class OBVIndicator {
  OBVResult calculate(List<KlineData> data) {
    if (data.isEmpty) return OBVResult(values: []);
    
    final obv = <double>[];
    double cumulative = 0;
    
    for (int i = 0; i < data.length; i++) {
      if (i == 0) {
        cumulative = data[i].volume.toDouble();
      } else {
        if (data[i].close > data[i - 1].close) {
          cumulative += data[i].volume;
        } else if (data[i].close < data[i - 1].close) {
          cumulative -= data[i].volume;
        }
      }
      obv.add(cumulative);
    }
    
    return OBVResult(values: obv);
  }
}
```

### 3.9 DMI（动向指标）

#### 3.9.1 指标含义

DMI（Directional Movement Index）由Welles Wilder发明，用于判断趋势的方向和强度。

**核心作用**：
- **+DI > -DI**：多头市场
- **+DI < -DI**：空头市场
- **ADX > 25**：趋势明确
- **ADX < 20**：震荡市场
- **ADX见顶回落**：趋势可能反转

#### 3.9.2 计算公式

```
步骤1：计算真实波幅(TR)
TR = max(high - low, |high - close(t-1)|, |low - close(t-1)|)

步骤2：计算方向运动(DM)
+DM = max(high - high(t-1), 0)，若low - low(t-1) > +DM则+DM = 0
-DM = max(low(t-1) - low, 0)，若high - high(t-1) > -DM则-DM = 0

步骤3：计算方向指标(DI)
+DI = 100 × EMA(+DM, N) / EMA(TR, N)
-DI = 100 × EMA(-DM, N) / EMA(TR, N)

步骤4：计算平均方向指数(ADX)
DX = 100 × |+DI - -DI| / (+DI + -DI)
ADX = EMA(DX, N)
```

#### 3.9.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期 | 14 | 6-30 | 计算周期 |

#### 3.9.4 Flutter实现

```dart
class DMIIndicator {
  final int period;  // 默认14
  
  DMIIndicator({this.period = 14});
  
  DMIResult calculate(List<KlineData> data) {
    if (data.length < period + 1) return DMIResult(plusDI: [], minusDI: [], adx: []);
    
    final tr = <double>[];
    final plusDM = <double>[];
    final minusDM = <double>[];
    
    // 计算TR和DM
    for (int i = 1; i < data.length; i++) {
      final trValue = max(
        data[i].high - data[i].low,
        max(
          (data[i].high - data[i - 1].close).abs(),
          (data[i].low - data[i - 1].close).abs(),
        ),
      );
      tr.add(trValue);
      
      final upMove = data[i].high - data[i - 1].high;
      final downMove = data[i - 1].low - data[i].low;
      
      double pdm = 0, mdm = 0;
      if (upMove > downMove && upMove > 0) {
        pdm = upMove;
      }
      if (downMove > upMove && downMove > 0) {
        mdm = downMove;
      }
      plusDM.add(pdm);
      minusDM.add(mdm);
    }
    
    // 计算EMA
    final smoothTR = _calculateEMA(tr, period);
    final smoothPlusDM = _calculateEMA(plusDM, period);
    final smoothMinusDM = _calculateEMA(minusDM, period);
    
    // 计算DI
    final plusDI = <double>[];
    final minusDI = <double>[];
    
    for (int i = 0; i < smoothTR.length; i++) {
      plusDI.add(100 * smoothPlusDM[i] / smoothTR[i]);
      minusDI.add(100 * smoothMinusDM[i] / smoothTR[i]);
    }
    
    // 计算DX和ADX
    final dx = <double>[];
    for (int i = 0; i < plusDI.length; i++) {
      final sum = plusDI[i] + minusDI[i];
      if (sum == 0) {
        dx.add(0);
      } else {
        dx.add(100 * (plusDI[i] - minusDI[i]).abs() / sum);
      }
    }
    
    final adx = _calculateEMA(dx, period);
    
    return DMIResult(plusDI: plusDI, minusDI: minusDI, adx: adx);
  }

}
```

### 3.10 DMA（平行线差指标）

#### 3.10.1 指标含义

DMA（Different of Moving Average）通过两条不同周期移动平均线的差值来判断趋势。

**核心作用**：
- **DDD上穿AMA**：金叉，买入信号
- **DDD下穿AMA**：死叉，卖出信号
- **DDD > 0**：短期均线在长期均线上方，多头市场
- **DDD < 0**：短期均线在长期均线下方，空头市场

#### 3.10.2 计算公式

```
DDD = MA(close, N1) - MA(close, N2)    // 短期均线 - 长期均线
AMA = MA(DDD, M)                        // DDD的移动平均

其中：
N1 = 短期周期（默认10）
N2 = 长期周期（默认50）
M = 平滑周期（默认10）
```

#### 3.10.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 短期周期 | 10 | 5-30 | 短期MA周期 |
| 长期周期 | 50 | 20-100 | 长期MA周期 |
| 平滑周期 | 10 | 5-30 | AMA周期 |

#### 3.10.4 Flutter实现

```dart
class DMAIndicator {
  final int shortPeriod;  // 默认10
  final int longPeriod;   // 默认50
  final int smoothPeriod; // 默认10
  
  DMAIndicator({
    this.shortPeriod = 10,
    this.longPeriod = 50,
    this.smoothPeriod = 10,
  });
  
  DMAResult calculate(List<KlineData> data) {
    if (data.length < longPeriod) return DMAResult(ddd: [], ama: []);
    
    final closes = data.map((d) => d.close).toList();
    
    // 计算短期MA
    final shortMA = _calculateMA(closes, shortPeriod);
    // 计算长期MA
    final longMA = _calculateMA(closes, longPeriod);
    
    // 计算DDD
    final ddd = <double>[];
    for (int i = 0; i < longMA.length; i++) {
      ddd.add(shortMA[i + (longPeriod - shortPeriod)] - longMA[i]);
    }
    
    // 计算AMA
    final ama = _calculateMA(ddd, smoothPeriod);
    
    return DMAResult(ddd: ddd, ama: ama);
  }
  
  List<double> _calculateMA(List<double> data, int period) {
    final ma = <double>[];
    for (int i = period - 1; i < data.length; i++) {
      double sum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        sum += data[j];
      }
      ma.add(sum / period);
    }
    return ma;
  }
}
```

### 3.11 BBI（多空指标）

#### 3.11.1 指标含义

BBI（Bull and Bear Index）通过多条不同周期移动平均线的平均值来判断多空分界。

**核心作用**：
- **价格 > BBI**：多头市场，持有
- **价格 < BBI**：空头市场，观望
- **价格上穿BBI**：买入信号
- **价格下穿BBI**：卖出信号

#### 3.11.2 计算公式

```
BBI = (MA3 + MA6 + MA12 + MA24) / 4

其中：
MA3 = 3日移动平均
MA6 = 6日移动平均
MA12 = 12日移动平均
MA24 = 24日移动平均
```

#### 3.11.3 参数配置

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 周期1 | 3 | 2-10 | 第一条MA周期 |
| 周期2 | 6 | 3-15 | 第二条MA周期 |
| 周期3 | 12 | 5-30 | 第三条MA周期 |
| 周期4 | 24 | 10-60 | 第四条MA周期 |

#### 3.11.4 Flutter实现

```dart
class BBIIndicator {
  final List<int> periods;  // 默认[3, 6, 12, 24]
  
  BBIIndicator({this.periods = const [3, 6, 12, 24]});
  
  BBIResult calculate(List<KlineData> data) {
    final maxPeriod = periods.reduce(max);
    if (data.length < maxPeriod) return BBIResult(values: []);
    
    final closes = data.map((d) => d.close).toList();
    
    // 计算各周期MA
    final maList = <List<double>>[];
    for (final period in periods) {
      maList.add(_calculateMA(closes, period));
    }
    
    // 计算BBI
    final bbi = <double>[];
    for (int i = 0; i < maList.last.length; i++) {
      double sum = 0;
      for (int j = 0; j < maList.length; j++) {
        final offset = maxPeriod - periods[j];
        sum += maList[j][i + offset];
      }
      bbi.add(sum / periods.length);
    }
    
    return BBIResult(values: bbi);
  }
  
  List<double> _calculateMA(List<double> data, int period) {
    final ma = <double>[];
    for (int i = period - 1; i < data.length; i++) {
      double sum = 0;
      for (int j = i - period + 1; j <= i; j++) {
        sum += data[j];
      }
      ma.add(sum / period);
    }
    return ma;
  }
}
```

## 4. 指标对比与选择建议

### 4.1 按类型分类

| 类型 | 指标 | 适用场景 |
|------|------|----------|
| **趋势类** | MACD, DMI, DMA, BBI | 判断趋势方向和强度 |
| **超买超卖类** | KDJ, RSI, WR, CCI | 判断买卖时机 |
| **波动类** | BOLL | 判断价格波动区间 |
| **能量类** | VOLUME, OBV | 衡量资金流向 |

### 4.2 组合使用建议

| 组合 | 优势 | 适用场景 |
|------|------|----------|
| **MACD + KDJ** | 趋势+超买超卖 | 综合判断买卖点 |
| **BOLL + RSI** | 波动区间+超买超卖 | 震荡市场操作 |
| **MACD + OBV** | 趋势+资金流向 | 确认趋势真实性 |
| **DMI + CCI** | 趋势强度+偏离程度 | 趋势市场操作 |
