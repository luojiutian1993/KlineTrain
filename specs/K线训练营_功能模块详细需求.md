## 二、首页模块 (Home)

### 2.1 模块概述

首页是用户进入App后的第一屏，承担 **内容分发、入口导流、用户引导** 的核心职责。

### 2.2 功能架构

```
首页 (HomePage)
├── 2.2.1 顶部导航栏
├── 2.2.2 搜索功能区
├── 2.2.3 快捷入口区

### 2.2.1 顶部导航栏

**功能描述：** 首页顶部状态栏，包含应用Logo、标题和快捷操作按钮。

**UI元素：**
| 元素 | 类型 | 说明 |
|------|------|------|
| Logo | 图标 | 品牌标识，点击跳转首页 |
| 标题 | 文本 | "K线训练营"，固定显示 |
| 消息入口 | 图标按钮 | 带红点未读提示，点击跳转消息列表 |
| 设置入口 | 图标按钮 | 点击跳转设置页面 |

**状态定义：**
- 默认状态：正常显示
- 消息红点：根据未读消息数量动态显示/隐藏
- 网络异常：显示断网图标

### 2.2.2 搜索功能区

**功能描述：** 提供品种搜索功能，支持模糊匹配股票/期货名称和代码。

**UI元素：**
```
┌─────────────────────────────────────────┐
│ 🔍 搜索股票/期货名称或代码              │
└─────────────────────────────────────────┘
```

**搜索范围：**
- A股股票（沪市、深市）
- 期货品种（商品期货、金融期货）

**交互逻辑：**
| 步骤 | 行为 |
|------|------|
| 1 | 用户点击搜索框，弹出搜索页面 |
| 2 | 用户输入关键字（至少1个字符） |
| 3 | 实时展示匹配结果（前10条） |
| 4 | 点击结果项，跳转品种详情页 |

**搜索结果展示：**
```
┌─────────────────────────────────────────┐
│ 贵州茅台  600519.SH  上海A股  白酒      │
├─────────────────────────────────────────┤
│ 泸州老窖  000568.SZ  深圳A股  白酒      │
└─────────────────────────────────────────┘
```

**字段说明：**
| 字段 | 说明 |
|------|------|
| 名称 | 品种中文名称 |
| 代码 | 交易所代码 |
| 市场 | 上海A股/深圳A股/期货 |
| 行业 | 所属行业（股票） |

**技术要求：**
- 搜索接口：GET `/api/v1/instruments?keyword={keyword}`
- 响应时间：< 200ms
- 支持拼音首字母搜索（后端实现）

└── 3.2.5 训练历史页
```

#### 3.2.3.1 K线图表区

**功能描述：** 展示历史K线数据，支持交互操作。

**图表配置：**
| 配置项 | 值 |
|--------|-----|
| 主图类型 | K线蜡烛图 |
| 均线 | MA5(黄)、MA10(紫)、MA20(蓝)、MA30(白) |
| 副图默认 | 成交量 |
| 副图可选 | MACD、KDJ、RSI、BOLL |

**K线样式：**
| 状态 | 颜色 | 说明 |
|------|------|------|
| 阳线 | 红色 (#FF3B30) | 收盘价 > 开盘价 |
| 阴线 | 绿色 (#34C759) | 收盘价 < 开盘价 |
| 十字星 | 灰色 | 收盘价 = 开盘价 |

**手势交互：**
| 手势 | 行为 |
|------|------|
| 左右滑动 | 切换K线区间 |
| 捏合缩放 | 调整显示K线数量（10-500根） |
| 长按 | 显示十字光标和数据浮层 |
| 双击 | 重置缩放比例 |

**数据揭示逻辑：**
- 训练开始时，只显示K线框架，不显示蜡烛体
- 用户点击"揭示下一根"按钮，逐步揭示K线
- 揭示后，该根K线变为完整显示
- 已揭示区域不可回退

**UI元素：**
```
┌─────────────────────────────────────────┐
│ [日/周/月] [指标▼]           [揭示下一根] │
└─────────────────────────────────────────┘
```

#### 3.2.3.3 持仓信息区

**功能描述：** 展示当前训练的持仓状态和盈亏情况。

**数据字段：**
| 字段 | 说明 |
|------|------|

#### 3.2.3.4 交易操作区

**功能描述：** 提供买入、卖出、换股等交易操作入口。

**按钮布局：**
```
┌─────────────────────────────────────────┐
│  [换股]  [条件单]  [买入(绿)]  [卖出(红)] │
└─────────────────────────────────────────┘
```

**换股功能：**
- 点击弹出品种选择器
- 确认后重置当前会话的品种

**买入/卖出功能：**
- 点击弹出交易对话框（详见3.2.3.4.1）
- 交易执行后刷新持仓信息

**状态约束：**
| 按钮 | 可用条件 |
|------|----------|
| 买入 | 可用资金 ≥ 最小买入金额 |
| 卖出 | 持仓数量 > 0 |
| 换股 | 持仓数量 = 0 |
| 条件单 | 始终可用 |

```
presentation/
├── providers/           # Riverpod Providers
│   ├── auth_provider.dart
│   ├── training_provider.dart
│   ├── instruments_provider.dart
│   └── user_provider.dart
├── pages/              # 页面
├── widgets/            # 组件
└── models/             # 数据模型
```

### 5.3 API接口清单

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/home/hot-training` | GET | 热门训练数据 |
| `/api/v1/home/recommendations` | GET | 推荐品种 |
| `/api/v1/instruments` | GET | 品种列表/搜索 |
| `/api/v1/training/sessions` | POST | 创建训练会话 |
| `/api/v1/training/sessions/{id}` | GET | 获取会话详情 |
| `/api/v1/training/sessions/{id}/reveal-next` | POST | 揭示下一根K线 |
| `/api/v1/trading/trades` | POST | 执行交易 |
| `/api/v1/trading/conditional-orders` | POST | 创建条件单 |
| `/api/v1/user/profile` | GET/PUT | 用户信息 |
| `/api/v1/user/favorites` | GET/POST/DELETE | 自选管理 |
