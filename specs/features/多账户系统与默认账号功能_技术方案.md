# 多账户系统与默认账号功能 - 技术方案

## 1. 概述

### 1.1 设计目标

基于需求文档 [多账户系统与默认账号功能.md](./多账户系统与默认账号功能.md)，实现：
1. 默认账号 `13071103531` 静默自动登录
2. 多账户数据完全隔离
3. 我的页面数据与当前账户同步
4. 重复注册检测、密码登录、密码重置

### 1.2 技术选型

| 技术点 | 选型 | 说明 |
|--------|------|------|
| 状态管理 | Riverpod | 现有技术栈，已使用 userProvider/authProvider |
| 路由 | go_router | 现有技术栈，已配置 AppRoutes |
| 数据库 | Drift | 现有技术栈，已配置 TrainingDao/UserDao |
| 日志 | appLogger | 共享工具，用于调试和问题排查 |

## 3. 模块设计

### 3.1 Auth Provider 与 CurrentUserNotifier

**文件**: `lib/providers/auth_provider.dart`

**核心组件**:

```dart
// 登录状态 Provider
@riverpod
class AuthState extends _$AuthState {
  @override
  bool build() => false;

  // 静默自动登录（13071103531）
  Future<bool> silentLogin() async {
    appLogger.d('[AuthState] silentLogin: 开始静默登录...');
    try {
      final userRepo = UserRepository();
      final user = await userRepo.getUserByPhone('13071103531');
      
      if (user != null) {
        state = true;
        ref.read(currentUserNotifierProvider.notifier).setUser(user);
        return true;
      }
      return false;
    } catch (e) {
      return false;
    }
  }

  // 手动登录
  Future<void> login(String phone, String password) async {
    final userRepo = UserRepository();
    final isValid = await userRepo.verifyPassword(phone, password);
    
    if (isValid) {
      final user = await userRepo.getUserByPhone(phone);
      if (user != null) {
        state = true;
        ref.read(currentUserNotifierProvider.notifier).setUser(user);
      }
    } else {
      throw Exception('手机号或密码错误');
    }
  }

  // 退出登录
  Future<void> logout() async {
    state = false;
    ref.read(currentUserNotifierProvider.notifier).clearUser();
  }

  // 注册（带重复检测）
  Future<bool> register(String phone, String password, String email) async {
    final userRepo = UserRepository();
    
    // 检查手机号是否已注册
    final existingUser = await userRepo.getUserByPhone(phone);
    if (existingUser != null) {
      throw Exception('该手机号已注册，请直接登录或找回密码');
    }

    final newUser = UserModel(...);
    await userRepo.saveUserToDatabase(newUser, password: password);
    state = true;
    ref.read(currentUserNotifierProvider.notifier).setUser(newUser);
    return true;
  }
}

// 当前用户 Provider
@riverpod
class CurrentUserNotifier extends _$CurrentUserNotifier {
  @override
  UserModel? build() => null;

  Future<void> setUser(UserModel user) async {
    state = user;
  }

  Future<void> clearUser() async {
    state = null;
  }
}
```

**日志覆盖点**:
- `silentLogin()`: 开始、查找结果、成功/失败
- `login()`: 开始、验证结果、成功/失败
- `logout()`: 开始、清除完成
- `register()`: 开始、重复检测、注册结果
- `setUser()`/`clearUser()`: 用户变化

### 3.2 MainTabPage 初始化逻辑

**文件**: `lib/features/main/main_tab_page.dart`

```dart
class _MainTabPageState extends ConsumerState<MainTabPage> {
  bool _initialized = false;

  @override
  void initState() {
    super.initState();
    WidgetsBinding.instance.addPostFrameCallback((_) {
      _initAuth();
    });
  }

  Future<void> _initAuth() async {
    if (_initialized) return;
    _initialized = true;

    final authState = ref.read(authStateProvider);
    if (!authState) {
      final success = await ref.read(authStateProvider.notifier).silentLogin();
      if (!success && mounted) {
        context.go(AppRoutes.login);
      }
    }
  }
}
```

## 4. 数据隔离方案

### 4.1 隔离层次

| 层次 | 隔离方式 | 说明 |
|------|----------|------|
| 用户层 | userId | users 表主键 |
| 会话层 | sessionId + userId | training_sessions 表 |
| 持仓层 | sessionId | positions 表（session 有 userId） |
| 交易层 | sessionId + userId | trades 表 |
| 条件单层 | sessionId | conditional_orders 表 |

### 4.2 查询约束

所有数据查询必须携带 `userId` 约束：

```dart
// TrainingDao 已有方法（无需修改）
Future<List<TrainingSession>> getUserSessions(int userId, ...)
Future<Map<String, dynamic>> getUserAssetSummary(int userId)
```

## 5. 重复注册检测

### 5.1 实现逻辑

```dart
Future<bool> register(String phone, String password, String email) async {
  final userRepo = UserRepository();
  
  // 1. 检查手机号是否已存在
  final existingUser = await userRepo.getUserByPhone(phone);
  if (existingUser != null) {
    // 2. 已存在，抛出异常
    throw Exception('该手机号已注册，请直接登录或找回密码');
  }

  // 3. 不存在，创建新用户
  final newUser = UserModel(...);
  await userRepo.saveUserToDatabase(newUser, password: password);
  state = true;
  ref.read(currentUserNotifierProvider.notifier).setUser(newUser);
  return true;
}
```

## 6. 密码重置

### 6.1 实现逻辑

```dart
Future<bool> resetPassword(String phone, String newPassword) async {
  final userRepo = UserRepository();
  
  final user = await userRepo.getUserByPhone(phone);
  if (user == null) {
    throw Exception('该手机号未注册');
  }

  await userRepo.updatePassword(user.id, newPassword);
  return true;
}
```
