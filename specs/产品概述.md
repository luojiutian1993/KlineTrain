# 产品概述

## 1. 项目愿景 (Vision)
> 成为国内领先的K线模拟训练平台，帮助初级交易员通过历史真实K线训练提升盘感，降低真实交易的学习成本和风险。

## 2. 核心价值 (Value Proposition)

*   **用户价值**：
    *   **降低学习门槛**：无需真金白银即可进行真实市场环境的模拟训练
    *   **提升交易技能**：通过反复练习K线形态识别，提高对短期走势的判断能力
    *   **建立交易纪律**：通过条件单（止盈止损、网格交易）训练良好的风控习惯
    *   **数据驱动改进**：基于训练记录统计，清晰了解自身交易风格和短板

*   **商业/技术价值**：
    *   **差异化竞争**：参考券商成熟的条件单模式，提供专业级模拟训练体验
    *   **用户粘性**：特殊指标和进阶功能规划（第二代产品），打造长期用户留存
    *   **技术可扩展**：Flutter跨平台架构，支持快速迭代和全平台覆盖

## 3. 核心板块 (Core Modules)

### 3.1 首页模块
*   **描述**：提供选股条件、市场概览、推荐标的等功能，帮助用户快速找到训练目标
*   **关键特性**：
    *   选股条件筛选（行业、市值、技术指标等）
    *   市场热门标的推荐
    *   快速开始训练入口
    *   今日训练统计概览

### 3.2 实战模块
*   **描述**：核心训练模块，基于历史真实K线数据进行模拟交易训练
*   **关键特性**：
    *   K线逐步揭示机制（先开盘价 → 再完整K线）
    *   自由买入/卖出交易
    *   条件单系统（止盈止损、条件买卖、网格交易）
    *   多周期切换（日线/周线/月线）
    *   训练结果实时统计

### 3.3 我的模块
*   **描述**：用户账户管理、个人训练记录和系统设置
*   **关键特性**：
    *   用户注册/登录（手机号+验证码）
    *   训练历史记录查询
    *   个人训练数据统计（胜率、盈亏比、最大回撤）
    *   账户与安全设置
    *   离线数据管理

## 4. 目标用户 & 核心场景 (User & Scenes)

### 4.1 目标用户

*   **[股票/期货新手]**：刚进入市场，缺乏实战经验，希望先模拟练习再入场
    *   核心诉求：低风险学习、熟悉K线形态、理解市场波动

*   **[有亏损经历的初级交易员]**：有一定交易经验但胜率不高，希望提升盘感
    *   核心诉求：找出交易短板、训练止损纪律、提高盈亏比

*   **[自我驱动的学习者]**：希望通过系统化训练形成自己的交易体系
    *   核心诉求：数据化复盘、量化自身进步、验证交易策略

### 4.2 核心场景 (Key Use Cases)

*   **场景 1**：[新手用户] 在通勤路上，打开APP选择"茅台"，系统随机从历史数据中选取2019-2021年区间，用户通过点击"下一步"逐步揭示K线，在价格回调时买入，在达到止盈位后自动卖出，最终查看本次训练的盈亏报告。

*   **场景 2**：[有经验的交易员] 选择自己关注的期货品种"沪铜"，设置网格交易单（上限12000、下限80000、10格），启动自动训练模式，系统自动执行网格买卖，用户观察策略效果并调整参数。

*   **场景 3**：[自我学习者] 在"我的"模块查看近30天的训练统计，发现自己的止损单触发率低但单次亏损金额大，据此调整交易习惯，重点训练持仓管理和止损执行。

## 5. 关键业务流程 (Core Workflows)

### 核心流程：K线训练会话

```
1. [用户选择标的] → 系统展示可选品种列表（A股/期货）
        ↓
2. [系统随机选取历史区间] → 从该品种近5年数据中随机选取连续2年
        ↓
3. [展示初始K线状态] → 显示训练起始日期和初始资金（默认10万）
        ↓
4. [训练循环开始]
   ┌─────────────────────────────────────────┐
   │  用户点击"下一步"                        │
   │         ↓                               │
   │  揭示下一根K线开盘价（Open价格显示）      │
   │         ↓                               │
   │  用户决策：                              │
   │  - 买入 / 卖出 / 观望                   │
   │  - 设置/修改/撤销条件单                  │
   │         ↓                               │
   │  用户再次点击"下一步"                    │
   │         ↓                               │
   │  揭示完整K线（High/Low/Close）           │
   │         ↓                               │
   │  系统检查条件单触发：                    │
   │  - 止盈止损 → 自动执行交易              │
   │  - 条件单 → 满足条件则执行              │
   │  - 网格单 → 触及网格线则执行            │
   │         ↓                               │
   │  更新账户状态（资金、持仓、盈亏）        │
   └─────────────────────────────────────────┘
        ↓
5. [训练结束条件触发]
   - K线走到选取区间末尾
   - 用户主动结束训练
   - 资金耗尽或亏损达50%
        ↓
6. [生成训练报告]
   - 总交易次数、胜率
   - 总盈亏金额、盈亏比
   - 最大单次盈利/亏损
   - 最大回撤
   - 条件单触发统计
```

## 6. 验收标准 (Acceptance Criteria)

*   **功能完备性**：
    *   用户可完成完整的模拟训练流程（选择标的 → 揭示K线 → 交易 → 查看报告）
    *   条件单系统可正常工作（止盈止损、条件买卖、网格交易）
    *   账户系统可用（注册、登录、保存训练记录）
    *   无崩溃、卡死等严重Bug

*   **技术指标**：
    *   K线图表渲染流畅（60fps）
    *   页面加载时间 < 2秒
    *   API响应时间 < 500ms
    *   iOS端可通过App Store审核上架

*   **合规要求**：
    *   应用内显著位置显示免责声明
    *   明确标注"模拟训练，不构成投资建议"
    *   不涉及任何真实证券交易

## 7. 约束条件 (Constraints)

*   **平台约束**：第一期仅开发iOS端（Flutter架构便于后续扩展Android/Mac/Windows）
*   **网络约束**：必须联网使用（数据实时从服务器获取）
*   **账户约束**：必须注册登录，游客不可使用任何功能
*   **数据约束**：仅支持历史数据训练，不支持实时行情（避免用户误解为真实交易）
*   **合规约束**：产品定位为教育工具，非证券交易软件，不需相关金融牌照
*   **开发周期**：MVP版本控制在10周内完成

## 8. 项目边界 (Out of Scope)

*   **暂不支持的功能**：
    *   真实交易（与券商对接）
    *   实时行情数据
    *   资讯/新闻/基本面数据
    *   自选股同步
    *   社交功能（排行榜、对战）
    *   社区/论坛

*   **暂不支持的平台**：
    *   Android（第二期规划）
    *   Mac桌面端（第三期规划）
    *   Windows桌面端（第三期规划）
    *   Web端

*   **暂不支持的市场**：
    *   港股、美股（第一期仅支持A股和商品期货）

*   **第二代产品功能**（与当前MVP无耦合）：
    *   AI辅助分析
    *   特殊技术指标组合
    *   资金流向/情绪分析
    *   策略回测功能
