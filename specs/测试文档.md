## 1. 概述

本文档描述 K线训练营App 的测试策略、测试用例和测试流程。

## 2. 测试策略

### 2.1 测试类型

| 测试类型 | 描述 | 优先级 |
|---------|------|--------|
| 单元测试 | 测试单个函数、方法或类的行为 | 高 |
| Widget测试 | 测试Flutter Widget的UI交互和渲染 | 高 |
| 集成测试 | 测试多个组件或模块之间的交互 | 中 |
| 端到端测试 | 模拟用户真实操作流程 | 中 |

### 2.2 测试覆盖率目标

- 单元测试覆盖率：≥ 70%
- Widget测试覆盖率：≥ 50%
- 核心业务逻辑：100% 覆盖

## 7. 测试执行命令

```bash
# 运行所有测试
flutter test

# 运行单元测试
flutter test test/unit/

# 运行Widget测试
flutter test test/widget/

# 运行集成测试
flutter test test/integration/

# 生成测试覆盖率报告
flutter test --coverage
genhtml coverage/lcov.info -o coverage/html

# 运行特定测试文件
flutter test test/unit/providers/kline_provider_test.dart
```

## 8. CI/CD集成

### 8.1 GitHub Actions配置

```yaml
name: CI

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main, develop ]

jobs:
  test:
    runs-on: ubuntu-latest
    
    steps:
      - name: Checkout code
        uses: actions/checkout@v4
        
      - name: Set up Flutter
        uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.41.x'
          
      - name: Install dependencies
        run: flutter pub get
        
      - name: Run tests
        run: flutter test
        
      - name: Generate coverage
        run: flutter test --coverage
        
      - name: Upload coverage
        uses: codecov/codecov-action@v4
        with:
          files: coverage/lcov.info
```
