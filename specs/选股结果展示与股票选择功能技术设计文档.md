# 选股结果展示与股票选择功能技术设计文档

## 概述

本文档描述 KlineTrain 项目选股结果展示、股票选择、跳转到实战功能的技术实现方案。

## 功能需求

| 需求编号 | 功能描述 | 优先级 | 实现状态 |
|:---:|------|:---:|:---:|
| REQ-HOME-009 | 筛选出符合条件的股票后，在首页展示具体股票信息列表 | 高 | ✅ 已实现 |
| REQ-HOME-010 | 在选股结果列表中选择某一只具体股票 | 高 | ✅ 已实现 |
| REQ-HOME-011 | 点击确认后跳转到实战页面，并传递选中股票信息 | 高 | ✅ 已实现 |

### 3.2 股票列表项样式

```dart
class StockListItem extends StatelessWidget {
  final Stock stock;
  final bool isSelected;
  final VoidCallback onTap;
  
  @override
  Widget build(BuildContext context) {
    return ListTile(
      // 选中状态
      selected: isSelected,
      leading: isSelected 
        ? CircleAvatar(child: Icon(Icons.check_circle))
        : null,
      
      // 股票信息
      title: Text(stock.name),
      subtitle: Text('${stock.marketCode}${stock.symbol}'),
      
      // 价格信息
      trailing: Column(
        crossAxisAlignment: CrossAxisAlignment.end,
        children: [
          Text(
            '¥${stock.lastPrice?.toStringAsFixed(2)}'),
          Text(
            '${stock.change > 0 ? '+' : ''}${stock.change?.toStringAsFixed(2)}%',
            style: TextStyle(
              color: stock.change > 0 
                ? Colors.red 
                : Colors.green),
          ),
        ],
      ),
      
      onTap: onTap,
    );
  }
}
```

    // 验证是否选中股票

          marketCode: selectedStock.marketCode,
          closePrice: selectedStock.lastPrice,
          condition: _selectedCondition,

)

## 6. 时间范围选择修复

### 6.1 问题分析

**问题描述**：点击近1年、近3年、近5年没有反应。

**根本原因**：
- HomeScreen调用StockConditionSelector时未传递selectedTimeRange参数
- 缺少onTimeRangeChanged回调

### 6.2 修复方案

**HomeScreen更新**：
```dart
// 添加状态变量
StockTimeRange _selectedTimeRange = StockTimeRange.recent1Year();

// 更新组件调用
StockConditionSelector(
  selectedTimeRange: _selectedTimeRange,
  onTimeRangeChanged: (timeRange) {
    setState(() => _selectedTimeRange = timeRange);
    // 重新选股
    _executeStockFilter();
  },
  // ...
)
```
