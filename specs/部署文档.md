# K线训练营App - 部署文档

**文档版本**: v1.0  
**最后更新**: 2026-05-13

## 1. 概述

本文档描述 K线训练营App 的构建和部署流程，包括各平台的构建配置和发布步骤。

## 2. 环境要求

### 2.1 开发环境

| 工具 | 版本 | 说明 |
|------|------|------|
| Flutter SDK | ≥ 3.41.0 | 核心开发工具 |
| Dart SDK | ≥ 3.2.0 | Dart语言支持 |
| Android Studio | ≥ Hedgehog | Android开发 |
| Xcode | ≥ 15.0 | iOS/macOS开发 |

| Node.js | ≥ 18.0 | 构建工具 |

```env

FIREBASE_PROJECT_ID=your-project-id
FIREBASE_STORAGE_BUCKET=your-storage-bucket
FIREBASE_MESSAGING_SENDER_ID=your-sender-id
```

## 3. 构建命令

### 3.1 通用命令

```bash
# 安装依赖
flutter pub get

# 代码生成（如有使用build_runner）
dart run build_runner build

# 运行开发服务器
flutter run

# 代码分析
flutter analyze

# 代码格式化
flutter format .
```

### 3.2 Android构建

```bash
# 调试版本
flutter build apk

# 发布版本
flutter build apk --release

# 指定架构
flutter build apk --release --target-platform android-arm64

# 构建App Bundle（推荐）
flutter build appbundle --release
```

### 3.3 iOS构建

```bash
# 进入iOS目录
cd ios

# 安装依赖
pod install

# 返回项目根目录
cd ..

# 调试版本
flutter build ios

# 发布版本（无签名）
flutter build ios --release --no-codesign

# 发布版本（有签名）
flutter build ios --release
```

### 3.4 macOS构建

```bash
# 添加macOS平台支持
flutter create . --platforms macos

# 安装依赖
cd macos && pod install && cd ..

# 调试版本
flutter run -d macos

# 发布版本
FLUTTER_DISABLE_GPU_SHADERS=true flutter build macos --release
```

**macOS构建注意事项**：

由于GPU着色器编译问题，macOS构建需要设置环境变量：
```bash
export FLUTTER_DISABLE_GPU_SHADERS=true
```

### 3.5 Windows构建

```bash
# 添加Windows平台支持
flutter create . --platforms windows

# 调试版本
flutter run -d windows

# 发布版本
flutter build windows --release
```

## 4. 发布流程

### 4.1 Android发布

**步骤1：生成签名密钥**

```bash
keytool -genkey -v -keystore ~/key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias key
```

**步骤2：配置签名**

在 `android/app/build.gradle` 中添加：

```gradle
android {
    ...
    signingConfigs {
        release {
            keyAlias 'key'
            keyPassword 'your-password'
            storeFile file('key.jks')
            storePassword 'your-password'
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

**步骤3：构建发布版本**

```bash
flutter build appbundle --release
```

**步骤4：上传到Google Play**

1. 登录 [Google Play Console](https://play.google.com/console)
2. 创建应用
3. 上传 `.aab` 文件
4. 填写应用信息
5. 提交审核

### 4.2 iOS发布

**步骤1：配置开发者账号**

1. 登录 [Apple Developer Portal](https://developer.apple.com/)
2. 创建App ID
3. 创建描述文件
4. 下载并安装证书

**步骤2：配置Xcode项目**

1. 打开 `ios/Runner.xcworkspace`
2. 选择项目 → Runner → Signing & Capabilities
3. 选择Team
4. 配置Bundle Identifier

**步骤3：构建发布版本**

```bash
flutter build ios --release
```

**步骤4：上传到App Store**

1. 打开Xcode → Window → Organizer
2. 选择归档文件
3. 点击 "Distribute App"
4. 选择 "App Store Connect"
5. 上传应用

### 4.3 macOS发布

**步骤1：配置签名**

1. 打开 `macos/Runner.xcworkspace`
2. 配置Signing & Capabilities
3. 添加 "App Sandbox" 权限

**步骤2：构建发布版本**

```bash
FLUTTER_DISABLE_GPU_SHADERS=true flutter build macos --release
```

**步骤3：打包DMG**

```bash
# 使用create-dmg工具
create-dmg \
  --volname "K线训练营" \
  --volicon "assets/icons/app.icns" \
  --window-pos 200 120 \
  --window-size 800 600 \
  --icon-size 100 \
  --icon "K线训练营.app" 200 300 \
  --hide-extension "K线训练营.app" \
  --app-drop-link 600 300 \
  "K线训练营.dmg" \
  "build/macos/Build/Products/Release/"
```

## 5. CI/CD配置

### 5.1 GitHub Actions

```yaml
name: CI/CD Pipeline

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main, develop ]

jobs:
  build:
    runs-on: macos-latest
    
    steps:
      - name: Checkout code
        uses: actions/checkout@v4
        with:
          fetch-depth: 0
      
      - name: Set up Flutter
        uses: subosito/flutter-action@v2
        with:
          flutter-version: '3.41.x'
          channel: 'stable'
      
      - name: Install dependencies
        run: flutter pub get
      
      - name: Run lint
        run: flutter analyze
      
      - name: Run tests
        run: flutter test
      
      - name: Build iOS
        run: flutter build ios --release --no-codesign
      
      - name: Build Android
        run: flutter build apk --release
      
      - name: Build macOS
        env:
          FLUTTER_DISABLE_GPU_SHADERS: true
        run: flutter build macos --release
```

### 5.2 环境变量配置

在GitHub Secrets中添加以下变量：

| 变量名 | 说明 |
|--------|------|
| ANDROID_KEYSTORE | Base64编码的keystore文件 |
| ANDROID_KEYSTORE_PASSWORD | keystore密码 |
| ANDROID_KEY_ALIAS | 密钥别名 |
| ANDROID_KEY_PASSWORD | 密钥密码 |
| APPLE_ID | Apple开发者账号 |
| APPLE_ID_PASSWORD | Apple账号密码 |
| APPLE_TEAM_ID | Apple团队ID |
